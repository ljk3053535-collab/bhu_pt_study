I. 관절질환 분류 & 특징 비교

▼
구분류마티스관절염(RA)골관절염(OA)
초기 침범부위윤활막관절연골
초기 침범관절MCP, PIP, 손목 (소관절 대칭)DIP, CMC (체중부하 비대칭)
무릎관절 변형밖굽이무릎안굽이무릎
강직아침강직 (30분↑, 휴식 후 악화)오후강직 (활동 후 악화)
결절없음헤버딘결절(DIP), 보챠드결절(PIP)
침범관절 특징부드러움(soft)딱딱함(hard), 염발음
검사소견RF+, ESR↑, CRP↑RF-, ESR 정상, CRP 정상
💡 통풍: 퓨린 대사장애 → 요산↑ → 요산염결정 침착 / 30~50대 남성 호발 / 엄지발가락 발작성 통증 / 야간에 심함
💡 혈우병성 관절염: 혈액응고인자 VIII·IX 결핍 / 반복적 관절 내 출혈 / 남성 호발

II. 관절 변형 정리

▼
변형명특징원인
단추구멍 변형PIP 굽힘 + DIP 과다폄류마티스관절염, 중앙 폄근힘줄 파열
백조목 변형PIP 과다폄 + DIP 굽힘류마티스관절염, 내재근 구축
Z형(엄지)MP 과다폄 + IP 굽힘자쪽곁인대 파열
망치손가락DIP 굽힘 변형DIP 폄근힘줄 파열
원숭이손엄지두덩 위축, 엄지 맞섬 소실정중신경 마비
갈퀴손(비숍손)MCP 폄 + IP 굽힘 (4·5번)자신경 마비
손목처짐손목관절 폄 불가노신경 마비

III. 관절질환 중재

▼
질환중재
류마티스관절염 급성기침상안정, ROM 유지, 부목·보조기
류마티스관절염 아급성·만성기ROM 운동, 근력강화, 기형 방지, 관절 보호
골관절염넙다리네갈래근 강화운동, 무릎 주변 근육 강화
통풍성 관절염급성기: 냉치료, 이온도입(리도카인·디클로페낙)
TKR(무릎전치환술) 후쪼그려 앉기 최소화 / 걷기·수영 권장
💡 RA 관절 보호: 가방끈이 있는 가방을 어깨로 걸쳐 메도록 함 (손가락 관절 보호)

IV. 연부조직 손상 & 구축

▼
인대 파열 등급특징
1도 (경도)섬유 몇 개 손상 / 관절 불안정성 없음
2도 (중등도)섬유 절반 손상 / 불안정성 약간
3도 (중증)섬유 모두 손상 / 불안정성 심함
구축 종류특징
근정적 구축근힘줄 단위 적응적 단축 / 원섬유 수 유지 / 뻗침으로 완화
가성 근정적 구축근육 과긴장(spasticity)으로 ROM 제한
섬유성 구축비가역적, 섬유화 변화

V. 근거중심 물리치료 & ICF

▼
💡 근거수준 피라미드 (신뢰도 ↓): 체계적고찰 > 무작위임상시험 > 코호트연구 > 환자-대조군연구 > 사례연구 > 전문가의견

PICO: Population(환자) / Intervention(중재) / Comparison(비교) / Outcome(결과) → 1단계 임상 질문 작성 시 사용

코호트 연구: 특정요인 노출 집단을 장기간 추적관찰 → 비노출 집단과 비교하여 비교위험도 제시

임상의사결정 과정검사 → 평가 → 진단 → 예후 → 중재
SOAP 노트S(주관적) → O(객관적) → A(평가/사정) → P(계획)
ICF 모델신체기능·구조 / 활동·참여 / 환경요인·개인요인

VI. PNF 기법 요약

▼
기법설명

수축-이완(CR)등장성 수축 후 이완 / ROM 변화 있음

안정적 반전주동근·길항근 교대 수축 / 균형·협응력↑
반복 신장반복적 빠른 신장 자극 / 운동능력·근력↑

⭐ VII. 시험 핵심 암기 포인트

▼
⭐ RA vs OA 핵심 구별:
RA = MCP·PIP 침범, 밖굽이무릎, 아침강직 30분↑, RF+, 부드러운 관절
OA = DIP 침범, 안굽이무릎, 헤버딘결절, RF-, 딱딱한 관절, 염발음

⭐ 손가락 변형 구별:
단추구멍 = PIP 굽힘+DIP 폄 (RA, 폄근힘줄 파열)
백조목 = PIP 폄+DIP 굽힘 (RA, 내재근 구축)
원숭이손 = 정중신경 마비 / 갈퀴손 = 자신경 마비 / 손목처짐 = 노신경 마비

⭐ 통풍 특징: 퓨린→요산↑ / 30~50대 남성 / 야간 발작성 / 이온도입(리도카인, 디클로페낙) / 급성기 냉치료
