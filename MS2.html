I. 관절가동범위운동 (ROM)

▼
종류정의적용대상제한점
PROM외부 힘으로 관절 움직임혼수, 마비, Poor 이하근위축 방지 ❌, 근력·지구력 증가 ❌
AROM근수축으로 스스로 움직임Fair 이상근력 유지·증진에 한계
AAROM완전 ROM 위해 보조 받는 AROM일부 수축 가능하나 Full ROM 어려울 때목적·적용대상·한계 = AROM과 동일
💡 CPM (지속적수동운동): 수술 직후 유착·구축 예방, 통증 감소 / 최대보호기(급성기)에 사용 / 근피로 없음 / 근력증진 ❌
💡 AAROM 보조도구: 파우더 보드(스케이트보드), 손가락 사다리, 교대적 운동기구, 머리 위 도르래

II. 뻗침운동 (Stretching)

▼
항목내용
적응증유착·구축·반흔으로 ROM 제한 / 근육 약하고 반대조직 단축
금기증골성 방해물 / 혈종·조직 외상 / 과운동성 / 구축이 관절 안정성 대신할 때 / 마비·부종
크립(Creep)장기간 낮은 부하 → 비수축성 조직 가소성 변형↑, 근원섬유마디 수↑
느린 뻗침조직 손상 최소화, 근방추 억제, GTO 활성 → 효과적
빠른 뻗침근방추 흥분 → 뻗침 반사 유발 → 보통 사용 ❌

스트레스-스트레인 곡선:

  • 탄성영역: 스트레스 제거 시 원래 길이로 복귀 (가역적)
  • 가소성(소성)영역: 스트레스 제거 후 원래 상태 ❌ (영구변형)
  • 항복점: 탄성↔가소성 경계 / 병목영역(Necking): 조직 약증, 단면적 감소
  • 최종파열점: 부분·완전 분리
💡 맨손뻗침 기법:
유지-이완(HR) = 등척성 수축 후 이완 / 관절가동범위 변화 없음 / 자가억제(GTO)
수축-이완(CR) = 등장성 수축 후 이완 / 관절가동범위 변화 있음
주동근수축기법(AC) = 반대방향 동심성 수축 / 상호억제 원리
