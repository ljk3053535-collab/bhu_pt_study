🔬 물질의 전자이론

분자: 물질의 성질을 지닌 최소 단위. 원자 → 분자 → 물질 순으로 구성됨.
전자: 음전하, 원자 궤도를 돌며 음 → 양으로 흐름 (전류 방향은 반대: 양→음).
전하: 단위 C(쿨롱). 1C = 1A가 1초 운반하는 전기량.
법칙공식 / 내용
옴의 법칙V = I·R (전류는 전압에 비례, 저항에 반비례)
줄의 법칙H = I²RT (발열은 전류 제곱에 비례)
쿨롱의 법칙전하량 곱에 비례, 거리 제곱에 반비례
패러데이 법칙전자유도 현상 → 유도전류 발생
전류 종류특징
전도전류도체 내 자유전자 흐름 → 자계·열 발생
이온화전류이온화액에서 이온 극성 따라 흐름
변위전류유전체(절연체)를 통한 전하 이동 → 유도 자기장 발생
⚡ 임상 포인트
전류밀도 감소법: ① 전극 접촉면 크게 ② 전극 간격 넓게 ③ 전류강도 낮추기
정전압 vs 정전류: 정전압 = 저항 변화 시 전류 변동 / 정전류 = 전압이 보상하여 전류 일정 유지

⚡ 의용전류 분류 & 효과

직류(DC): 1초 동안 한 방향으로 흐름. 화학적 효과가 주됨.
교류(AC): 전기강도·방향이 주기적으로 바뀜. 저·중·고주파로 구분.
분류주파수주요 효과종류
저주파1,000Hz 이하신경근 자극, 역학적 효과TENS, FES, SSP, HVGS
중주파1,000~100,000Hz역학적·통증완화·조직치유ICT, 러시안전류 (주로 4,000Hz)
고주파100,000Hz 이상심부투열(열효과만)단파, 극초단파, 초음파
🔴 양극(+) 효과
산성 반응 · 흥분성 ↓ · 역치 ↑
진정·통증감소
혈관수축 · 지혈효과
🔵 음극(-) 효과
알칼리 반응 · 흥분성 ↑ · 역치 ↓
자극·통증유발
혈관확장 · 출혈효과
화학적 효과
음극: 4Na + 4H₂O → 4NaOH + 2H₂ (수소 발생)
양극: 4Cl + 2H₂O → 4HCl + O₂ (산소 발생)
※ 기체 발생률: 음극(H₂) : 양극(O₂) = 2:1
⚡ 임상 포인트
전기진단: 감응전류(교류) → 정상근 / 단속평류전류(직류) → 마비근·변성근
전기화학적 화상: 양극에서 주로 발생 (산성 반응으로 조직 손상)

🧪 전기진단학

신경재생: 1일 1mm
변성반응: 신경손상 10일 후 나타남 (변성 완성까지 10~14일 소요)
→ 변성반응 없으면 2~4주 이내 회복 예상
검사정상값비고
평류강축비(GTR)3.5~6 : 1변성 10:1 / 탈신경 1.5~1:1 / 재생 1~20:1
기전류 검사2~18mA최소 가시 수축 유발 최소 전류강도
시치 검사1ms 이하기전류 2배로 흥분 일어나는 시간
신경재생 속도= 양 자극부위 간 거리(mm) ÷ 잠복시 차(ms)
단속직류 - 근육단속직류 - 신경감응전류 - 근육감응전류 - 신경
정상연축연축강축강축
부분변성반응감소느린수축반응감소반응감소
완전변성반응X느리고 완만한 수축반응X반응X
절대변성반응X반응X반응X반응X
강시곡선(SD-커브): 강도-시간 관계 곡선
→ 변성 심할수록 오른쪽 위로 이동 / 회복될수록 왼쪽 아래로 이동
→ 탈신경근: 우상향 곡선, 선이 매끄럽지 않고 불규칙
근전도(EMG)
· 안정 시 이상 근전도: 세동전위, 양성예파, 섬유속 연축, 고빈도반사
· 정상 근전도: 운동 단위 활동전위
⚡ 임상 포인트
변성반응 검사 시기: 손상 후 최소 10일 이후 실시 (변성 완성에 10~14일 필요)
정상근 단속직류 자극: 연축 후 빠른 이완
감응전류·단속직류로 신경자극 시 반응 모두 감소 → 부분변성

💪 근육섬유 & 전기생리학

섬유형특징수축속도피로
I형 (완근, 적색근)소운동신경 지배, 긴장근느림낮음
Ia형큰 운동신경 지배빠름중간
II형 (속근, 백색근)큰 운동신경 지배, 지구력↓가장 빠름높음
근수축 과정
탈분극 → 신경말단 활동전위 → 아세틸콜린 분비 → 종판전위 → 근육 섬유 활동전위 발생 → 근세포질그물 Ca²⁺ 방출 → 액틴·마이오신 결합 → 근수축
극성공식(정상근): CCC > ACC > AOC > COC
탈신경근: CCC < ACC < AOC < COC (반대!)

강축 자극빈도: 20~25pps → 강축 / 30pps 이상 → 근피로
⚡ 임상 포인트
연축 시간: 잠복기(0.01s) → 수축기(0.04s) → 이완기(0.05s)
안정막전위 -70mV: 안정 시 K⁺ 투과성 = Na⁺의 100배
절대불응기: 어떤 자극에도 반응 없음 / 상대불응기: 더 큰 자극이면 활동전위 발생

🧠 신경섬유 분류

섬유종류기능특징
Aα말이집근육 운동종판 날신경크고 빠름, 운동·감각
Aβ말이집촉각, 압각, 골지힘줄기관
Aγ말이집근방추 날신경
Aδ말이집빠른 통증·온도·경한 촉각
B말이집자율신경 절전섬유작고 느림
C민말이집느린 통증(작열통·둔통), 자율신경 절후가장 얇고 느림, 역치 가장 높음
⚡ 임상 포인트
굵기 순서: Aα → Aβ → Aδ → B → C (갈수록 얇고, 전도속도↓, 역치↑)
TENS 통증 조절 (관문조절설): Aβ 자극 → SG 활성화 → Aδ·C 섬유 통로 차단

⚙️ 전기치료기기 구성

기본 3가지: 전원공급회로 → 발진회로 → 출력증폭회로
· 발진회로: 교류에 진동 발생, 주파수·맥동기간·강도 등 변환
· 출력증폭회로: 강도 최종 조절·증폭
구성요소기능
변압기전자유도 작용으로 전압 효율적 변환
정류기교류 → 직류 또는 단상맥동전류 전환
여과기맥류 → 평류 변환
조정기부드러운 전류·전압 발생 조정
파형특징
정형파 전류방향·크기 대칭 변화 (TENS, SSP, FES, HVGS)
감응전류(패러딕)비대칭, 자극시간 1ms 고정 (정상 신경근 자극·진단)
격동전류세기가 갑자기 강약 반복
펄스파전압 높고 지속시간 짧음 (동통 최소화)
⚡ 임상 포인트
전선 종류: 전기코드(저전압) / 전도케이블(단파) / 동축케이블(극초단파·초음파)
누설전류: 치료 중 기구와 환자가 접촉 시 발생하는 원치 않는 전류
