⏱ 연령별 핵심 키워드 & 차이점
🏃 대동작 발달 4단계 (0~12개월) — 핵심만
1단계 0~3M★HEAD CONTROL
  • 핵심: 머리 정렬(head alignment)
  • 특징: Hypertonia (굴곡 우세)
  • Prone: 2M에 45° → 3M에 1분 들어올림
  • 3개월 = on-elbows posture 완성
  • 원시서기(primary standing) / 원시보행 가능
2단계 4~6M★SITTING UP
  • 핵심: 팔 지지에 의한 pushing up + sitting up
  • Prone: hand support → pivot prone(비행기 자세)
  • 4~5M 뒤집기 가능
  • 배밀이(creeping) 시작 — 배 바닥
  • 5M: propped sitting / bridging position
  • 낙하산 반응 출현 시작
3단계 7~9M★지속 이동
  • 핵심: 이동성 + 활동성
  • 7M: 네발기기 자세(quadruped) — rocking
  • 기기(crawling): 배 안 닿음 / creeping과 차이!
  • 7M: unsupported sitting 가능
  • 9M: 잡고 서기 / 잡고 이동(cruising)
  • pull up to standing
4단계 10~12M★걷기
  • 핵심: 독립 보행
  • 10M: W-sitting, side-sitting 가능
  • 10M: 능숙한 기기 / squat 앉기
  • 11M: 한 손으로 잡고 걷기
  • 12M: 혼자 서기 / 가구 잡고 걷기

✋ 소동작 발달 핵심 키워드
1단계 0~3M
  • 손움켜잡기반사(grasping reflex) 강함
  • 신생아: 수의적 잡기 못함
  • 1~2M: 짧게 잡을 수 있으나 바로 놓음
  • 3M: 물건 입으로 가져감 / 180° 추적
2단계 4~6M
  • 손 파악반사 소실
  • 4M 후반: 양손으로 물건 잡아 입으로
  • 5M: palmar grasp 가능
  • 6M: 두 물건 동시 쥠 / 손바닥 아치 발달
  • 6M: radial palmar grasp
3단계 7~9M
  • 7M: radial digital grasp
  • 엄지+다른 손가락 쥐기(prehension) 가능
  • 7~8M: 양손 물건 쥐기
  • 9M: 물건 놓기(release) 가능
4단계 10~12M
  • 10M: pincer grasp 가능
  • 10M: three jaw chuck
  • 11~12M: 능숙한 pincer grasp
  • 12M: finger tip pinch
14~60M (유아기)
  • 14M: 입방체 2개 쌓기
  • 24M: 입방체 6개 / 수직선 그리기
  • 그리기: 원→사각형→삼각형 (3→4→5세)
  • 가위질: 학령전기
⚡ 원시반사(Primitive Reflex) — 시험 핵심
★ 소실 시기별 분류 (중요!)

4개월 소실

  • 놀람반사(startle reflex)

📅 표 4-2 연령별 발달 — 영역별 핵심
생후 2개월
운동: ATNR / prone머리 돌림 인지: 애착형성 시작
  • 바로 누워 ATNR 자세 빈번
  • 2~6개월: 가까운 사람 선호 반응
  • social smile: 3~5주 → 보통 2개월 명확히 출현
  • 자기를 돌봐주는 사람 믿는 기본적 안정감 형성 (애착형성기)
  • 눈 발달: supine에서 천천히 추적
생후 2~6개월
head control 완성 3개월 고저조절★ 4M 소이 반응
  • 3M: 엎드린 자세 팔 뻗어 몸 지지 → head control
  • 4M: 전완 6로 지지 → 안면 드는데 스스로 못할 경우 sh. control 이상형
  • 3~4M: 비대칭 긴장성 몸 반사 소실 → 양손 모아 물건
  • 4~5M 뒤집기 가능
  • 4M: 주위 흥미 → 수유 중 고개 돌림 (소이 반응!)
  • 6M: 가족 애착 + 낯선 사람 불안감
생후 6~12개월
7M 혼자 앉기 8M 배밀이 9M 대상영속성 10M O-position
  • 7M: 혼자 앉기 / 9M: 허리 돌릴 수 있음
  • 8M: 배밀이(creeping) + 바로누운자세에서 잡아 일어서기
  • 12M: 혼자 걷기 (아이들 많은 편)
  • 10M: 손 움직임 발달 → 엄지+집게 물건 잡기 (O position)
  • 9M: 대상영속성 — 물건 가려져도 존재 알게 됨
  • 만지는 물건 떨어뜨린 후 안 보이면 찾아냄
  • 8M: 분리 불안 많이 발생
  • 돌이 되면 엄마/아빠 한다
생후 12~18개월
support 없이 계단 엄마 행동 흉내
  • 12M 혼자 걸음 / 15M 못 걷는 아이도 있음
  • 18M: 불안하지만 뛸 수 있음
  • 12M: 바이바이, 안돼, 주세요 등 간단한 지시 이해
  • 15M: 몇 개 단어로 표현 / 크레파스 잡고 선
  • 이 시기: 가족 구성원 행동 보며 흉내냄 (엄마 행동 흉내내기)
생후 18~24개월
2층 세트 / 계단 언어 급발달
  • 한손으로 계단 올라감 [2층]
  • 24M: 균형 유지 → 민첩하게 움직여 계단
  • 이 시기 언어 발달이 가장 특징적
  • 18M: 어휘 10~15개 / 24M: 100개 이상
  • 단어 조합 → 단순 문장 / 두 문장 말 이해
  • 18M: 인지 성장은 정서 + 언어발달과 부합
학령전기 (3~5세)
3세 불잡지 않고 걷기 5세 줄넘기 오이디푸스 4~5세
  • 3세: 불잡지 않고 계단 / 한발 서기 잠깐
  • 4세: 한발 들고 뜀
  • 5세: 한 발 교대 들고 뛰기 / 줄넘기 가능
  • 그리기: 원(3세) → 사각형(4세) → 삼각형(5세)
  • 언어: 2,000개 어휘 / 3M 의문형·진행형·소유격
  • 4세: 과거형 / 5세: 미래형
  • 3세: 독자성 확인 욕구 / 이름·성별·나이 말함
  • 4~5세: 오이디푸스(Oedipus) — 엄마에 특별 관심 / 아빠 경쟁심 ambivalence
  • 3~4세: 친구와 협조적 놀이
  • 4~5세: 규칙 따르지만 상황 따라 다름
  • 5세 이후: 일정 규칙에 따라 놀이
학령기
고기능의 힘에 기여 주용언어 발달★
  • 근육의 힘과 협응성 증가 → 고기능의 힘에 기여
  • 피아노·춤추기·농구공 등 복잡한 운동
  • 인지 발달: 6~12세 뚜렷 / 현실적 이해
  • 주용언어 발달 수준 → 집중력 영향 → 인지·언어 정상 발달해야 학교생활 OK
  • 정신: 성과에 따라 평가받는 것이 가장 중요
  • 학업 불진: 두통·복통 신체 증상 유발 가능
🧠 뇌성마비 (Cerebral Palsy) — 종류·기전·임상
정의 & 유병률
  • 비진행적 손상 → 운동·자세 장애 초래 활동 제한
  • 태아 or 영아 뇌에 발생 (2004년 정의)
  • 유병률: 2~2.5명/1,000명
  • 36주 이하 = 37주 미만 동일
  • 원인 비율: 출산 전 34% / 주산기 43% / 출생 후 6%
  • 주산기 원인이 가장 많다!
위험인자 (표 6-1)
  • 출생 전: 조산(36주 이하) / 저체중(2.5kg 이하) / 모체 감염(풍진) / 간질
  • 주산기: 임신중독증 / 조기양막파열 / 분만 중 질 출혈 / 저산소증
  • 출생 후: CNS 감염 / 저산소증 / 고빌리루빈혈증 / 머리부 외상
  • 미숙아(1.5kg 이하): 유병률 77명/1,000명으로 급증
관련 주요 질환 (3개 암기)
  • ① 고빌리루빈혈증 — 황달 → BBB 통과 → 핵황달(kernicterus) → 바닥핵·뇌줄기 침범
  • ② IVH (뇌실내출혈) — 32~34주에 배아기질 소멸 → 34주 이전 위험. 4단계 분류
  • ③ PVL (뇌실주위 백질연화증) — 32주 미만 미숙아 / 겉질척수로 손상 → 양하지마비 많음
IVH 4단계
  • I: 뇌실막·배아기질 국한
  • II: 뇌실로 확장 / 50% 미만
  • III: 뇌실 확장 동반
  • IV: 뇌실질까지 확장
  • 문제: 3,4단계에서 뇌손상 가능성
뇌성마비 분류 3가지
① 해부학적 (topography)
  • 단마비(monoplegia): 거의 없음
  • 반신마비(hemiplegia): 한쪽
  • 양하지마비(diplegia): 하지 심함
  • 삼지마비(triplegia)
  • 팔다리마비(quadriplegia): 사지 + 몸통
② 운동장애 형태
  • 강직형(spastic): 대뇌겉질 손상 / 70%
  • 운동이상형(dyskinesia): 바닥핵 손상
  • 실조형(ataxia): 소뇌 손상 / 저긴장
③ GMFCS (기능)
  • 1단계: 제한 없이 걷기
  • 2단계: 걷지만 제한적
  • 3단계: 보행보조기구 사용
  • 4단계: 자가이동 가능하나 제한
  • 5단계: 수동 휠체어로 다른 사람이 옮겨야

강직(Spasticity) 정의
  • "근육의 수동적 신장 시 나타나는 속도 의존적 저항"
  • 신장 속도↑ = 저항↑ (비례하지는 않음)
  • 감정·통증·접촉에 따라 증가 가능
  • 손상된 운동시스템(UMN) 관련
  • 반사항진(hyperreflexia) / 클로누스(clonus) 동반
★ 개정된 Ashworth 척도 (MAS)
  • 0: 근 긴장도 증가 없음
  • 1: 약간 증가 (잡힘 느껴졌다 사라짐)
  • 1+: 1/2 이하 범위에서 소 저항
  • 2: 대부분 범위에서 현저히 증가 / 쉽게 움직임
  • 3: 수동 움직임 어려움
  • 4: rigid (굽히거나 펼 때)
★ 개정된 Tardieu 척도 (MTS)
  • R1: 빠른 속도 신장 시 catch 각도
  • R2: 느린 속도 수동 관절 운동 각도
  • V1: 최대 느리게 / V2: 중력 / V3: 최대 빠르게
  • 근육반응 0~5 (5: 관절 못 움직임)
  • R2-R1 차이 = 동적 구축 정도
동반장애 비율
  • 지적장애: 30~65% (팔다리마비에 많음)
  • 간질성 경기: 25~45% → 수면 뇌파 검사
  • 시각장애: 40%에서 시각장애·안구조절 문제
  • 청각장애: 약 10% (연관성 그리 크지 않음)
  • 삼킴장애: 강직형 팔다리마비에 특히 많음
  • 심리적 문제: 불안·우울·과잉행동
이동 평가 — FMS (기능적 이동척도)
  • Rating 6: 모든 지면 독립
  • Rating 5: 평지 독립
  • Rating 4: 지팡이 (한쪽·양쪽)
  • Rating 3: 목발
  • Rating 2: 보행기·frame
  • Rating 1: 휠체어 (일어설 수 있음)
  • Rating C: 기기 (5m)
  • Rating N: 해당거리 걷기 불가
강직형 반신마비 보행 4유형 (Rodda 2001)
유형 1 — 처진발(drop foot)
  • 앞정강근 약화
  • hinged AFO / PLS AFO
유형 2 — 참꿈치들린발 (true equinus)
  • 장딴지근 강직·구축
  • 뛰기보행(jump gait) 가장 흔함
  • 관절형 AFO
유형 3 — 참꿈치들린발+뒤기무릎
  • 넙다리뒤근·넙다리곧은근 동시 수축
  • 무릎 ROM 45도 이하
유형 4 — 발목+무릎+골반 모두
  • 가장 복잡 / 양하지마비 양상
  • 고정형 AFO / 지면반발 AFO
강직형 양하지마비 보행 5유형 (Rodda 2004)
유형 1 — 참꿈치들린발
  • 장딴지근 / 관절형 AFO
유형 2A — 뛰기보행
  • 장딴지근+넙다리뒤근
  • 관절형 AFO
유형 3 — 겉보기꿈치들린발
  • 발목은 정상 / 무릎·고관절 굽힘
  • 고정형 AFO
유형 4 — 웅크림보행(crouch gait)
  • 가장 심각 / 에너지 효율 최저
  • 지면반발 AFO
유형 5 — 비대칭 보행
  • 반신마비+양하지마비 혼합
🔍 근골격계 주요 검사 키워드
Ely test (일라이 검사)
  • 목적: 넙다리곧은근(rectus femoris) 단축
  • 엎드린 자세 → 반대쪽 다리 빠르게 굽힘
  • 허벅지 바닥에서 떨어지면 → 강직·구축 ✓
Ogunlal angle / 모음근 단축 검사
  • 무릎 세우고 바로 누운 자세
  • 엉덩관절 벌림 → ROM 측정
  • 허벅지 고정 → 무릎 폄 시 모음 증가 = 안쪽 넙다리뒤근·두덩정강근 단축
오금각도(popliteal angle) 검사
  • 목적: 넙다리뒤근 단축
  • 바로 누운 자세 / 엉덩관절 90° 굽힘
  • 무릎 폄 시 넙다리뼈·정강뼈 각도 측정
Silverskiold test
  • 목적: 장딴지근 vs 가자미근 단축 구별
  • 무릎 폄 → 발등굽힘 제한 = 장딴지근
  • 무릎 굽힘 → 발등굽힘 잘 됨 = 가자미근은 OK
엉덩관절 돌림 검사
  • 엎드린 자세 무릎 굽히고 검사
  • 과도한 안쪽 돌림 → 넙다리뼈 앞경사(femoral anteversion) 증가
  • 과도한 가쪽 돌림 → 앞경사 감소
정강뼈 비틀림 검사 (2nd toe test)
  • 엎드린 자세 / 무릎 90° 굽힘
  • 두 번째 발가락 아래로 위치
  • 넙다리뼈 고정 → 수직선과 정강뼈 각도 측정
척주 변형 종류 & 관련 유형
  • 척주옆굽음증(scoliosis): 강직형 팔다리마비 + 긴장이상형 많이 발생
  • 척추뒤굽음증(kyphosis): 앉은자세 균형 떨어지는 아동
  • 허리 과다앞굽음증(hyperlordosis): 엉덩관절굽힘근 구축 + 보행 가능 아동
  • 골반 기울기: 척주옆굽음증 + 엉덩관절 탈구 관련
발 변형 종류
  • 안쪽들린발(pes varus): 앞정강근·뒤정강근 강직
  • 가쪽들린발(pes valgus): 종아리근·장딴지근 강직 / 무지외반
  • 굼치들린발(pes equinus): 반신마비 + 보행 불가 아동
  • 젖힌무릎(genu recurvatum): 장딴지근 단축으로 유발
