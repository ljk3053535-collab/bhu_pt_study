복장빗장관절 (SC Joint)

안장관절 · 자유도3 · 팔의 바닥관절

일반적 특징

  • 유형: 안장관절(Saddle)
  • 자유도: 3
  • 팔다리뼈대↔몸통뼈대 연결
  • 바닥관절(basilar joint)
  • 관절원반 있음 (사체 50%에서 완전)

관절주위 결합조직

  • 앞·뒤 복장빗장인대
  • 갈비빗장인대(CCL) → 올림 제한
  • 빗장사이인대(ICL) → 내림 제한
  • 목빗근, 복장방패근, 빗장밑근
SC관절 운동형상학
운동관절운동형상학제한 구조
올림 (35~45도)볼록→오목 위쪽 구르기+아래 미끄러짐갈비빗장인대(CCL)
내림 (10도)볼록→아래쪽 구르기+위쪽 미끄러짐빗장사이인대(ICL), 위관절주머니
내밈 (15~30도)오목→앞쪽 구르기+미끄러짐 동일방향뒤관절주머니, CCL 뒤다발
뒤당김 (15~30도)오목→뒤쪽 구르기+미끄러짐 동일방향앞관절주머니(ACL), CCL 앞다발
축돌림관절원반 가쪽면에 대한 복장끝의 스핀어깨 벌림·굽힘과 연계 / 독립 불가
💡 올림·내림 = 세로지름(볼록) / 내밈·뒤당김 = 가로지름(오목)

봉우리빗장관절 (AC Joint)

평면관절 · 부리빗장인대 · 탈구 주의

일반적 특징

  • 유형: 평면관절(Gliding)
  • 빗장뼈 가쪽끝 ↔ 어깨뼈 봉우리
  • 관절원반 있음 (형태 다양)

부리빗장인대 (Coracoclavicular Lig.)

  • 마름인대(Trapezoid): 앞가쪽 / 납작한 사각형
  • 원뿔인대(Conoid): 뒤안쪽 / 수직 / 원뿔 모양
  • AC관절의 가장 중요한 외재성 안정자
AC관절 운동 (돌림성 조절운동)
운동발생 면발생 상황
위쪽돌림 (최대 30도)이마면어깨 벌림·올림 시 일차적 AC운동
아래쪽돌림이마면팔 내려올 때
뒤기울임(Post. tilt)시상면어깨뼈 위쪽돌림 시 AC관절 기여
가쪽돌림(Ext. rotation)수평면어깨뼈 위쪽돌림 시 AC관절 기여
⚠️ AC관절 탈구
  • 대학 미식축구 선수 어깨 손상의 약 40%
  • 수평 전단력 → 위·아래관절주머니인대가 일차적 저항
  • 심한 경우 → 부리빗장인대가 이차적 저항 → 파열 시 탈구
  • 불안정성 → 외상후뼈관절염 유발 가능
💡 부리빗장인대 기전: 앞톱니근이 어깨뼈 위쪽돌림 → 부리빗장인대 팽팽해짐 → 크랭크처럼 빗장뼈 뒤쪽돌림 유발

어깨가슴관절 (ST Joint)

진짜 관절 아님 · SC+AC 합성운동 · 안정자세

어깨가슴관절은 진짜 관절이 아니라 어깨뼈 앞면과 가슴의 뒤-가쪽벽 사이의 접촉부위다. ST에서의 움직임 = SC관절과 AC관절의 협동 합성운동

어깨가슴관절 움직임 6가지
운동설명SC기여AC기여
올림어깨뼈 위쪽으로 미끄러짐빗장뼈 올림어깨뼈 아래쪽돌림
내림올림의 역방향빗장뼈 내림—
내밈안쪽모서리 앞-가쪽으로SC 내밈AC 안쪽돌림
뒤당김내밈의 역방향SC 뒤당김AC 가쪽돌림
위쪽돌림관절오목 위쪽 향함빗장뼈 올림AC 위쪽돌림
아래쪽돌림위쪽돌림의 역방향빗장뼈 내림AC 아래쪽돌림
어깨뼈 안정자세 (Resting Posture)
  • 앞기울임 약 10도
  • 위쪽돌림 약 5~10도
  • 안쪽돌림 약 30~40도 (어깨면과 일치)
✅ 어깨뼈 완전한 위쪽돌림의 3가지 기능적 중요성
  • ① 관절오목을 위쪽·앞-가쪽으로 위치 → 팔의 위쪽·가쪽 손뻗기 최대화 (구조적 바닥부)
  • ② 중간어깨세모근·가시위근 등 오목위팔관절 벌림근에 대한 최적의 길이-장력관계 유지
  • ③ 봉우리밑공간 유지 → 충돌·손상 방지

오목위팔관절 (GH Joint)

관절주머니 · 인대 · 정적·동적 안정성

일반적 특징
  • 관절주머니 공간 용적 = 위팔뼈머리 크기의 약 2배 → 넓은 가동범위
  • 아래 부분 = 겨드랑주머니(axillary pouch) – 느슨한 오목(recess)
GH관절 인대 (관절주머니인대)
인대팽팽해지는 동작중요성
위오목위팔인대가쪽돌림, 아래·앞 병진운동아래방향 탈구 저항
중간오목위팔인대45~90도 벌림+가쪽돌림앞쪽 병진운동 저항
아래오목위팔인대 (가장 중요)90도 벌림+완전 가쪽돌림 (앞섬유)
90도 벌림+완전 안쪽돌림 (뒤섬유)		벌림 시 앞방향 탈구 일차적 저항
부리위팔인대팔 옆에 있을 때아래방향 탈구 저항
부리봉우리인대부리돌기↔봉우리부리봉우리 아치 형성 / 위쪽 탈구 방지
가로인대결절사이고랑 위두갈래근 긴갈래 힘줄 유지
💡 봉우리밑공간(Subacromial Space) = 위팔뼈머리와 부리봉우리활 사이 / 건강한 성인 약 1 cm / 내부: 가시위근힘줄, 봉우리밑윤활주머니, 두갈래근 긴갈래
정적 안정자 (Static Stabilizer)
구조기능
오목위팔관절주머니인대가동범위 끝 지점에서 최대 안정성
오목테두리(Glenoid labrum)관절오목 깊이 약 50% 기여 / 접촉면적↑
관절속 음압위팔뼈머리 흡착 유지
정적 잠김기전(SCS)위관절주머니+부리위팔인대+가시위근힘줄 → 잠김압박력(CF) 발생
💡 동적 안정자 = 돌림근띠(RC) → 관절이 어떤 위치에 있든 능동적 안정성 제공
GH관절 운동형상학
운동관절운동형상학특이사항
벌림위쪽 구르기 + 아래쪽 미끄러짐AHD 최소 = 20~35도 (2.6mm)
굽힘위팔뼈머리의 스핀 (일차적)최소 120도 / 완전굽힘=GH+ST위쪽돌림
가쪽돌림뒤쪽 구르기 + 앞쪽 미끄러짐60~70도 / 90도벌림 시 거의 90도
안쪽돌림앞쪽 구르기 + 뒤쪽 미끄러짐75~85도
💡 동적 중심화: 구르기 + 미끄러짐의 협동 / 뒤관절주머니 타이트 → 위팔뼈머리 앞쪽이동 / 앞관절주머니 타이트 → 뒤쪽이동

어깨위팔리듬 & 완전 벌림 6원리

Scapulohumeral Rhythm · 2:1 · 6가지 운동형상학적 원리

어깨위팔리듬 = 2 : 1  |  GH 120도 + ST 60도 = 180도
완전한 어깨 벌림 6가지 운동형상학적 원리
원리내용관절
원리 12:1 리듬 → GH 120도 + ST 60도 동시 발생GH + ST
원리 260도의 어깨뼈 위쪽돌림 = AC관절 위쪽돌림 + SC관절 올림 결합AC + SC
원리 3벌림 동안 빗장뼈는 SC관절에서 뒤당김SC
원리 4완전 벌림 동안 어깨뼈는 뒤기울임 + 약간 가쪽돌림AC
원리 5빗장뼈는 자신의 축에 대해 뒤쪽으로 돌림 (20~35도)SC 축돌림
원리 6GH관절은 가쪽으로 돌림 → 큰결절이 봉우리밑공간 통과GH
각 관절별 수치 (그림 5.36)

SC관절

  • 올림: 25도
  • 뒤당김: 20도
  • 뒤쪽돌림(축돌림): 25도

AC관절

  • 위쪽돌림: 30도
  • 뒤기울임: 20도
  • GH 가쪽돌림: 35~40도
💡 원리 5 기전: 앞톱니근 수축 → 어깨뼈 위쪽돌림 → 부리빗장인대 팽팽 → 크랭크처럼 빗장뼈 뒤쪽돌림
💡 원리 6 이유: 가쪽돌림 없으면 큰결절이 봉우리 아래 높은 지점에 충돌 / 대부분 벌림 70~80도 전에 완료

어깨가슴관절 근육

올림 · 내림 · 내밈 · 뒤당김 · 위쪽돌림 · 아래쪽돌림

기능주요 근육핵심 포인트
올림근 위등세모근, 어깨올림근, 큰·작은마름근 위등세모근 = SC관절에 대한 좋은 지레작용
내림근 아래등세모근, 넓은등근, 작은가슴근, 빗장밑근 목발보행(CKC): 아래등세모근+넓은등근 수축 필수
내밈근 앞톱니근 (가장 중요) SC 수직 돌림축 중심 / push-up plus 작용
뒤당김근 중간등세모근, 마름근들, 아래등세모근 마름근+아래등세모근 힘선 결합 = 순수 뒤당김
위쪽돌림근 (짝힘) 앞톱니근(SA) + 위등세모근(UT) + 아래등세모근(LT) SA = 가장 큰 모멘트팔 / 가장 효율적인 위쪽돌림근
아래쪽돌림근 큰·작은마름근, 작은가슴근 —
위쪽돌림 짝힘 (Force-couple) 분석 (그림 5.49·5.50)
짝힘결과 운동
앞톱니근(SA) + 아래등세모근(LT)AC관절 돌림축 중심 뒤기울임
앞톱니근(SA) + 중간등세모근(MT)AC관절 돌림축 중심 가쪽돌림
SA + UT + LT 전체 짝힘어깨뼈 위쪽돌림 생성
💡 앞톱니근 마비(긴가슴신경 손상) → 날개어깨뼈(winged scapula) / 머리 위로 팔 올리기 불가 / 마비 85%는 우세팔에서 발생
💡 "둥근 어깨(Rounded Shoulders)": 어깨뼈 내림·아래쪽돌림·내밈 → SC·GH관절 스트레스 → 봉우리밑공간 내 조직 끼임

📌 동적 중심화 (Dynamic Centering, Special Focus 5.5)
  • 돌림근띠 근육 = 구르기 + 미끄러짐을 협동 조절하여 위팔뼈머리를 관절오목에 중심화
  • 가시아래근 수축 → 뒤쪽 구르기 / 앞관절주머니인대(ACL) 신장 → 앞쪽 미끄러짐 유도
  • 과도한 뒤관절주머니 타이트함 → 가쪽돌림 시 위팔뼈머리 앞쪽이동 → 봉우리밑충돌 위험
  • 과도한 앞관절주머니 타이트함 → 위팔뼈머리 뒤쪽이동 → 불안정성
돌림근띠증후군 (Special Focus 5.9)
  • 가시위근힘줄의 과도한 마모·열상·염증
  • 원인: 반복된 부딪힘, 노화, 외상, 과사용
  • 가시위근 MA = 약 1:20 → 부하 무게보다 20배 더 큰 힘 필요
  • 임상: 물체를 신체 가까이에서 잡도록 권장 (외적 모멘트팔 감소)

어깨 최대 토크 순서 (그림 5.57)
폄 > 모음 > 굽힘 > 벌림 > 안쪽돌림 > 가쪽돌림
안쪽돌림근 vs 가쪽돌림근

안쪽돌림근 (IR)

  • 어깨밑근, 큰가슴근, 넓은등근, 큰원근, 앞어깨세모근
  • 가쪽돌림근보다 약 40~70% 더 큰 토크
  • 팔 30도 올림 시 어깨밑근 = 가장 큰 모멘트팔

가쪽돌림근 (ER)

  • 가시아래근, 작은원근, 뒤어깨세모근
  • 어깨 근육무리 중 가장 낮은 최대 토크
  • 기능적으로 매우 중요 (GH 동적 안정성)
  • 가쪽돌림 없으면 벌림 시 큰결절 충돌
📌 뒤어깨세모근 (Special Focus 5.10)
  • 폄근 + 모음근 + 가쪽돌림근 + 수평벌림근 동시에
  • 강한 수축 시 아래등세모근에 의해 어깨뼈가 견고하게 안정되어야 함
  • 활 쏘기 동작 = 뒤어깨세모근과 아래등세모근의 협력 관계 잘 나타냄
  • 겨드랑신경 과신장 → 완전 마비 가능 → 폄·수평벌림 어려움

임상 요약

어깨뼈 운동장애 · 봉우리밑충돌 · 유착성관절주머니염 · 재활

어깨뼈 운동장애 (Scapular Dyskinesis)
  • 정의: 원인에 관계없이 어깨뼈의 비정상적 위치나 움직임
  • 흔한 임상 표현: 감소된 위쪽돌림 / 과도한 아래쪽돌림 / 앞쪽돌림 / 앞기울임
  • "발음성 어깨뼈(snapping scapula)": 가슴에 대한 어깨뼈의 삐걱소리
봉우리밑 부딪힘 증후군 (Subacromial Impingement Syndrome)
  • 원인: 벌림 시 아래쪽 미끄러짐 부족 → 봉우리밑공간 내 반복 압박
  • 손상 조직: 가시위근 힘줄, 봉우리밑윤활주머니, 두갈래근 긴갈래
  • 기여 요인: 어깨뼈 위쪽돌림 부족 / 뒤관절주머니 타이트함 / RC 근육 약화
유착성관절주머니염 (Adhesive Capsulitis)
  • 아래관절주머니인대의 과도한 뻣뻣함 → 벌림 시 아래쪽 미끄러짐 부족 → 부리봉우리활에 걸림
보존적 치료 목표
  1. 관절에 대한 일차적·이차적 스트레스 감소
  2. 관절운동형상학의 정상화
  3. 능동·수동 가동범위의 회복
  4. 근력 증진
  5. 통증·염증 감소
