🧠 1. 뇌졸중 평가도구
BRS (브룬스트롬 회복단계) ★★
1단계 이완성 단계, 수의운동 불가
2단계 경직 서서히 발달, 연합반응, 기본 공동운동
3단계 경직 최강, 공동운동 수의적 가능, 갈고리집기
4단계 경직 감소, 공동운동 분리 시작, 가쪽집기
5단계 공동운동 상실, 원통·공 잡기
6단계 경직 없음, 개별 관절운동, 협조운동 정상
Fugl-Meyer 평가

  • 총 226점, 6개 영역, 3점 척도 (0·1·2점)
  • 팔66+다리34+균형14+감각24+ROM44+통증44=226점
  • 소항목별 3회 실시 → 최고점 채택

편측무시 평가

직선이등분검사 (LBT)

  • 18개 직선 → 중앙에 수직선
  • 정상: ¼인치(6.3mm) 미만
  • 경한무시: ¼인치 이상
  • 심한무시: ½인치(12.5mm) 이상

알버트검사 (Albert Test)

  • A4용지 40개 선, 모두 찾아 표시
  • 중앙 4개 제외, 36개 확인
  • 36개 찾으면 시야제한 없음
뇌졸중 결손 척도
NIHSS: 초기 장해·신경학적 상태 추적 / 11항목
MRS (수정된 Rankin): 0~6단계
0=증상없음 / 3=약간도움 중등도 / 4=보행불가 / 5=침상의존 / 6=사망

UPDRS (단일화된 파킨슨병 계측척도)
  • 총 199점, 4개 영역 (Part I~IV), 42문항, 0~4점
  • Part I: 의식·행동·기분 / II: ADL / III: 운동검진 / IV: 치료부작용
  • 점수 높을수록 심각
PDQ-39 (파킨슨 질문지)
  • 삶의 질 평가 / 8영역 39항목 / 총 100점 만점
  • 5점 리커트 척도 / 점수 높을수록 건강상태 나쁨
헌팅톤병 - UHDRS
  • 30분 이내 다도구 평가
  • 6영역: 운동(15개)·인지(6개)·행동(11개)·기능(21개)·독립척도·전체기능
