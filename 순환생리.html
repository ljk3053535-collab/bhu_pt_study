I. 물질의 이동 ▼
1. 수동운반 (passive transport) — 에너지 불필요 ▾
종류구동력이동 대상대표 예시
단순 확산농도차용질 (인지질 직접 통과)O₂/CO₂ 가스교환, 지용성 물질
촉진 확산농도차수송단백질 이용Na⁺·K⁺ (전압개폐채널), 인슐린→포도당
삼투농도차물(용매)적혈구 용혈, 탈수 시 혈장삼투압↑
여과압력차용질+물 모두사구체여과, 모세혈관→조직액
단순확산 촉진 조건: 극성 없음 / 지질 용해도 큼 / 분자 크기 작음 / 온도↑ / 농도차↑
수송단백질 종류: ① 통로 단백질 (채널) — 통로 역할  ② 운반체 단백질 — 결합 후 구조변화로 운반
삼투압: 반투과성 막이 받는 압력 / 농도차 클수록 큼 / ATP 사용 X
2. 능동 수송 — 에너지(ATP) 필요 ▾
  • 저농도 → 고농도 이동 (농도차를 거슬러 이동)
  • 세포막 운반체 단백질이 관여
  • Na-K 펌프: Na⁺ 3개 → 밖 / K⁺ 2개 → 안
  • 세뇨관 포도당 재흡수 / 소장 포도당·아미노산 흡수
3. 용적 운반 ▾
종류방향특징예시
세포내 이입밖→안능동적, 운반체단백질 무관
① 포식세포 작용 (고형 물질)
② 포음세포 작용 (액체)		백혈구의 세균 포식
세포외 유출안→밖분비소낭이 세포막과 융합인슐린·글루카곤 분비
아세틸콜린 방출

2. 체액의 이동 — 스타링 가설 ▾
힘정의크기방향
액압(HP)모세혈관내 혈액 압력세동맥끝 35 / 세정맥끝 15 mmHg혈관 → 조직
혈장교질삼투압알부민이 물을 끌어당기는 힘25 mmHg조직 → 혈관
조직압(IFP)조직 사이질 공간 압력0 ~ −1 mmHg혈관 → 조직 (도움)
조직교질삼투압조직 단백질이 물 당기는 힘약 2 mmHg혈관 → 조직
세동맥끝: 액압(35) > 혈장교질삼투압(25) → 여과(혈관→조직)
세정맥끝: 액압(15) < 혈장교질삼투압(25) → 재흡수(조직→혈관)
3. 체액 조절 이상 ★기출★ ▾
탈수 (49회 13번)

② 혈장콜로이드삼투압 감소 (혈장단백질↓) → 수분 유출
③ 모세혈관 투과성 증가 (염증) → 단백질 혈관 밖 누출
④ 림프관 폐쇄 → 조직액 회수 불가

2. 혈장 & 혈장단백질 ▾
  • 혈액의 55% (물 90%, 단백질 7~9%, 무기염류 1%, 포도당 0.1%)

  • 섬유소원 7% — 혈액응고 (간 생산)
  • 글로불린 30% — 면역기전 (림프구 생산)
  • 혈청: 혈장에서 피브리노겐이 제거된 상태

IV. 면역 ▼
1. 비특이적 면역 (1차 방어, 선천성) ▾
  • 병원체 종류 구분 없이 동일한 방식으로 신속·광범위하게 작용
  • 외부 방어벽: 피부(각질층, 라이소자임, 피지샘 산성) / 점막(섬모, 위산)
  • 내부 방어: 포식작용(큰포식세포) / 염증반응(히스타민→혈관확장·투과성↑→백혈구 이동)
