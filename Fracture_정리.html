골절(Fracture) 완전 정리 노트

근골격계 물리치료학
I. 개요 II. 분류 III. 증상 IV. 치유 V. 치유인자 VI. 뼈유합 VII. 합병증 VIII. 부위별 IX. 검사 X. 중재
I 개요 (Overview) ▾
정의
  • 뼈에 가해지는 힘이 내구성 초과 → 뼈 손상 또는 선상 변형
  • 뼈의 연속성이 완전 또는 부분적으로 파괴된 상태
발생 요인
  • 외적 요인: 외력의 형태, 크기, 방향, 속도
  • 내적 요인: 뼈 자체의 감수성 (골밀도, 연령, 대사질환, 강도)
II 골절의 분류 (Classification) ▾
1) 골절에 따른 분류

완전골절 — 양쪽 피질 연속성 완전히 끊어짐 / 주로 성인

  • 감입골절: 골절된 뼈 한쪽 끝이 다른 쪽 뼈 속에 박힘
  • 합병골절: 주위 중요 조직·장기·기관 손상 동반
  • 압박골절: diaphysis 골절 없이 metaphysis가 diaphysis 속으로 감입·압박
  • 개방골절: 골편이 피부 뚫고 외부 노출 감염↑
  • 단순골절: 골절편 수 3개 미만
  • 분쇄골절: 2개 이상 골절선, 골절편 3개 이상

불완전골절 — 한쪽 피질만 손상 / 주로 소아

  • ★생목골절: 소아 호발, 한쪽 부러지고 반대쪽 휘어짐 골절선O
  • 균열골절: 전위 없이 금만 가고 구조적 연속성 유지
  • 관통골절: 큰 힘이 작은 면적 → 뼈 관통
  • 함몰골절: 주로 두개골, 아래로 함몰
2) 골절 방향에 따른 분류
  • 가로상(transverse): 장축에 직각 / 직접력 / 광범위 물렁조직 손상 / 안정성
  • 경사상(oblique): 압박+구부러짐+비틀림 복합력 / 짧고 둥근 골절선 / 불안정성
  • 나선상(spiral): 비틀림력 / 골절선 길고 예각 / 불안정성
  • 세로상(longitudinal): 골절선이 뼈 장축을 따라 형성
3) 피부손상 유무에 따른 분류
  • 폐쇄골절: 골절부가 피부 밖으로 노출 안 됨
  • 개방골절: 골편이 피부 뚫고 노출 → 혈종이 외부 공기에 오염 → 감염↑ → 치료 시 특별 주의
4) 골절편 수에 따른 분류
  • 단순골절: 골절편 3개 미만 / 물렁조직·피부 손상 없음 / 전위 적음
  • 분쇄골절: 2개 이상 골절선, 골절편 3개 이상
  • 분절골절: 별도의 2개 완전골절이 하나의 뼈 안에 동시 존재
5) 골절편 전위에 따른 분류
  • 각형성전위: 뼈에서 먼 방향으로 힘 / 반대쪽 긴장
  • 엇갈림전위: 평행한 방향으로 수평 전위 / 정렬 어려움
  • 돌림전위: 축성 회전 / 정복 어려움
  • 신연전위: 골편이 당겨져 분리 / 골유합 저해
  • 겹침전위: 뼈 길이 단축
  • 끼임전위: 주로 골단부, 끼임 고정
6) 해부학적 위치에 따른 분류
  • 뼈끝(epiphysis) / 뼈몸통끝(metaphysis) / 뼈몸통(diaphysis)
  • 몸쪽(proximal) / 중간(middle) / 먼쪽(distal) 1/3
  • 관절 내(intra-articular) / 관절 외(extra-articular)
  • 넓적다리뼈: 머리, 목, 돌기, 돌기아래, 관절용기위 골절 등
7) 손상기전에 따른 분류
외상
  • 감입골절: 축성 부하·압박력 → 뼈잔기둥(trabecula) 압박
  • 찢김골절: 장력 부하 → 근육 수축/인대 저항 → 뼈 조각 떨어짐 / 돌출부에 호발
병적 과정
  • 구조적으로 약해진 뼈에 작은 외상으로 쉽게 발생
  • 원인: 골다공증, 파제트병, 골종양, 골감염 등
비정상적 스트레스
  • 피로골절: 정상 뼈에 작은 스트레스 반복 → 뼈아교질 연속성 중단
  • 예: 장거리 행군 → 발허리뼈 / 육상선수 → 정강뼈
III 골절의 일반적 증상 ▾
  • 통증과 압통: 골절 시 통증/압통 / 척추 경미한 압박골절은 통증 미미
  • 기능 소실: 지레작용 상실 + 물렁조직 손상 / 피로·압박골절은 일부 유지
  • 변형: 전위 or 물렁조직 손상 → 각형성·돌림변형
  • 부기: 물렁조직·혈관 손상 → 국소 부기 → 점차 확대
  • 자세의 변화: 의식적·무의식적 골절부 보호 반응
  • 비정상적 움직임·마찰음: 골편 부딪쳐 마찰음 → 추가 손상 주의
  • 신경·혈관 동반 손상: 반드시 검사 / 위팔뼈·넓적다리뼈 관절용기위 골절 시 필수
  • 피부색의 변화: 출혈·물렁조직 손상 → 반상출혈(ecchymosis)
IV 골절의 치유 (Healing Process) ▾
긴뼈(long bone) = 위팔뼈, 노뼈, 자뼈, 넓적다리뼈, 정강뼈, 종아리뼈 / 치유과정은 연속적·중복적으로 진행
① 염증기 (Inflammatory Phase) — 3~14일, 전체 10%
골절 발생 직후 ~ 연골이나 뼈 형성될 때까지 / 뼈바탕질 내 세포·혈관 손상 + 골막·물렁조직 손상 → 혈종(hematoma) 형성 → 핏덩이(blood clot) → 염증 유도체 유리 → 혈관 확장·혈장 삼출 → 급성 부종 / 대식세포·림프구 이동 → 괴사조직 제거 → 섬유모세포(fibroblast) 출현 → 복원기로 이행
② 복원기 (Reparative Phase) — 1~10개월, 전체 40%
혈종의 기질화(organization) / 핵심 세포: 중간엽세포(mesenchymal cell) → 골형성세포·연골세포로 분화 / 골모세포(osteoblast) 관여 / 골절부에 애벌뼈(callus) 형성 / ★이 시기의 뼈유합은 완성 단계 아님 — 재형성기에 접어들어야 정상 강도
③ 재형성기 (Remodeling Phase) — 수개월~수년, 전체 70%
미숙뼈·연골 → 성숙 층판뼈로 대치 / 파골세포(osteoclast) 출현 → 과형성된 애벌뼈 흡수 / 힘의 방향에 따라 뼈잔기둥이 더 치밀하게 배치
해면뼈 골절의 치유과정
  • 긴뼈 골절 치유과정과 유사
  • 혈종 형성 → 혈종 사이로 신생 혈관 들어가 골형성세포 증식
  • 골절된 양쪽면에서 같은 양상으로 성장 → 서로 접촉 → 석회화 → 격자뼈 형성
V 골절 치유의 기간에 영향을 미치는 인자 ▾
  • 나이: 젊을수록 빠름 / 소아는 급속 치유 (간엽→골형성세포 분화 빠름)
  • 호르몬: 촉진(성장호르몬, 갑상선호르몬, 칼시토닌, 인슐린) / 억제(부신피질호르몬)
  • 감염: 정상 조직 괴사, 종창, 혈관내 혈전 → 지연
  • 손상의 정도: 경미 → 조기 유합 / 심함 → 골형성세포·혈관 증식 지연
  • 혈액질환: 전신쇠약·영양불량 → 지연 / 넓적다리뼈머리·손배뼈·목말뼈 = 혈행차단 잘 됨
  • 만성 소모성 질환: 당뇨병, 활동성 결핵 → 지연
  • 신경마비: 생리적 전기자극 감소 → 지연 / 단, 중추신경 손상 시 근경련 → 오히려 촉진
  • 개방골절·분쇄골절: 물렁조직 소실·혈행장애·혈종 → 지장
  • 뼈 괴사: 혈행 차단 → 모세혈관 증식에 의해서만 유합 진행 → 지연
치유 빠른 조건: 소아·젊은 연령 / 근육에 둘러싸인 부위 > 피하 부위 / 해면뼈 > 피질골 / 뼈몸통끝 > 뼈몸통 / 접촉 면적 크고 전이 적음 / 폐쇄골절 > 개방골절 / 팔골절 > 다리골절
VI 뼈유합의 분류 ▾
부정유합 (malunion) ★잘못 붙음
  • 유합은 됐으나 원래 해부학적 위치가 아닌 상태로 유합
  • 각형성변형, 돌림변형, 뼈 길이 단축
  • 9세 미만 각형성 → 자연 교정 가능 / 돌림변형은 자연 교정 안 됨
  • 원인: CNS 손상 경련성 마비, 부정확한 맞춤, 고정 불충분, 심한 물렁조직 손상
  • 골절부 재정렬 후 재교정 필요
지연유합 (delayed union) ★혈액공급 부족
  • 충분한 기간(5개월) 치료했음에도 유합이 지연
  • 치유과정이 완전히 정지된 건 아님 → 계속 치료 시 유합 가능
  • 원인: 전신질환, 혈액순환 장애, 고정·맞춤 불완전, 고정기간 불충분, 영양불량
불유합 (nonunion) ★가성관절
  • 뼈유합기전이 정지되어 유합 가능성이 없는 상태
  • 9개월 이상 치료에도 뼈유합 진행 없거나 3개월간 방사선 변화 없을 때
  • 방사선상 골절부 양쪽 끝이 둥글고 진하게 됨, 골절선 선명
  • 호발: 정강뼈 아래 1/3, 노뼈 먼쪽, 넓적다리뼈 목, 위팔뼈 중간, 손배뼈
VII 합병증 (Complications) ▾
  • 감염: 개방성 창상 통한 직접감염이 대부분 / 성인 골수염 70%가 골절 후 발생
  • 무혈성 괴사: 골절·탈구로 혈류 차단 / 대표: 넓적다리뼈 목·머리, 손배뼈 골절의 몸쪽부
  • 외상성 관절염: 관절 내 골절이 해부학적으로 맞춰지지 않음 → 관절연골 마모
  • 관절경직: 오랜 석고붕대 고정 → 근육·관절막 위축·섬유화 / 석고붕대 고정 원칙: 골절부 위아래 관절 포함
  • 혈관손상: 큰 혈관 파열·폐쇄 / 전체 혈관 막히면 괴저 / 일부 막히면 허혈성 구축(ischemic contracture)
  • 근육과 힘줄의 손상: 콜레스골절 합병증 → 긴엄지폄근힘줄(EPL) 파열
  • 신경손상: 해당 신경 지배 부위에 감각마비·감각이상
  • 외상 후 골위축 (Sudeck's atrophy): 교감신경 흥분상태 / 방사선상 점상 골다공증(spotty osteoporosis) 특징 / 임상 3단계: 초기(따끔거림·발적) → 이영양기(차고 번질거림) → 위축기(근육·피부 위축)
  • 내장기관 손상: 골반·가슴 골절 시 내장기관 손상
★ CRPS (복합부위통증증후군): 외상 후 비정상적 통증, 감각 변화, 부기, 혈관운동 불안정성, 관절 강직성 / 팔 > 다리 (55:45) / 중년기 여성에 호발 / 치료: 조기 발견·진단 + 적극적 통증 조절 + 물리치료
VIII 부위별 골절 ★부위마다 다름 ▾
팔골절 — 위팔뼈

위팔뼈 몸쪽골절 (proximal humerus fracture)

  • 골다공증 있는 노인·여성에 호발 / 팔 짚고 넘어질 때
  • 분류: Neer 분류 (이분·삼분·사분 골절 + 골절탈구)
  • 합병증: 팔신경얼기 손상 (60%), 혈관 손상, 골화성근염, 무혈성 괴사

위팔뼈 중간골절 (humerus shaft fracture)

  • 청장년층 / 직접·간접 외력
  • ★Holstein-lewis증후군: 나선상골절 맞춤 시 노신경이 골절편 사이에 끼어 마비
팔골절 — 팔꿈치 관절

팔꿈치머리골절 (olecranon fracture)

  • 합병증: 골화성근염(이소성 골화증) — 팔꿈치(위팔근) 호발 / 예방이 최선 / Collar and Cuff sling 사용
  • ★볼크만허혈성 구축 (Volkmann's): 아래팔부 앞쪽 구획증후군
  • 5P sign: Pain, Pallor, Pulselessness, Paresthesia, Paralysis
  • → 정중신경·자신경 괴사 → 내재근 마비 → 갈퀴손가락 변형(claw hand)

운반각 (carrying angle)

  • 정상: 약 15° (남 8.9°, 여 11~12°)
  • 과도한 가쪽휜팔꿈치(cubitus valgus): 15° 이상
  • 안쪽휜팔꿈치(cubitus varus): 5° 이하 = 개머리판 변형(gunstock deformity)
  • 팔꿈치관절 골절 합병증 중 가장 높은 빈도
팔골절 — 아래팔뼈·손목뼈
  • 몬테지아골절: 자뼈 몸쪽골절 + 노뼈머리 탈구 동반 / 뼈사이신경 마비 흔함
  • 갈레아찌골절: 노뼈 먼쪽골절 + 아래노자관절 탈구 동반 / = 역몬테지아 = 필요골절
  • 자뼈단독골절: = 야경봉골절(nightstick fracture)
  • 콜레스골절: 손바닥 짚고 넘어짐 / 노뼈 먼쪽 골절 / 포크형 변형(dinner fork deformity) / 골다공증 중년 여성 / 합병증: 정중신경염, 긴엄지폄근힘줄 파열
  • 스미스골절: 손등으로 짚고 넘어짐 / = 역콜레스골절 / 정원삽 변형(garden spade deformity)
  • 손배뼈골절: 젊은 성인 남자 호발 / 허리골절이 가장 多 / 무혈성 괴사 자주 발생 / 검사: ★타진검사(percussion test)
팔골절 — 손골절
  • 망치손가락골절(mallet finger): DIP관절에 강한 굽힘력 → 손가락폄근 먼쪽 부착점 단열 → 굽힘 변형 / = 야구손가락 / 심하면 백조목 변형(swan neck deformity)
  • 복서골절(boxer fracture): 5번째 손허리뼈목골절 / = 짜움꾼골절 / 자뼈고랑부목으로 70~90° 고정
  • 베네트골절(Bennett's): 1번째 손허리뼈 몸쪽 + 노뼈쪽 탈구 / ★2조각 / 관절내 골절
  • 롤란도골절(Rolando's): 엄지손가락 손허리뼈 바닥부 분쇄골절 / ★Y가 / Y형 배치의 관절내 골절
몸통골절

빗장뼈골절

  • 소아·청소년에 자주 발생 / 가쪽 1/3과 중간 1/3 지점 사이에서 호발
  • 치료: 팔걸이 or 팔자형 석고붕대(figure 8 cast)

척주골절

  • ★고리뼈골절 = 제퍼슨골절(Jefferson's): C1 / 장축 방향 외력 / 골편이 바깥쪽으로 벌어짐 / ★골절 시 척수손상은 일어나지 않음
  • ★치아돌기골절 제2형: 가장 흔함 / 불안정 / 불유합 32~36%
  • ★교수형골절(hangman fracture): C2 과다폄+수직압박 → 척수 절단 → 사망

갈비뼈골절

  • 주로 5~9번째에서 발생
  • ★1~3번째: 팔신경얼기 손상 가능
  • ★10~12번째: 간·비장·신장 열상 가능
다리골절 — 골반·엉덩관절
  • 골반골절 합병증: 출혈(사망의 주요 원인) / 요도손상 가장 흔함 → 협착·발기부전·요실금
  • 갭핑검사(gapping test): ASIS에 대고 아래·가쪽 방향으로 힘 가함 → 통증 유발 시 골반골절 의심
  • 넓적다리뼈목골절: 노년층 / 골다공증 + 넘어질 때 / 합병증: 무혈성 괴사(15~40%), 불유합(전위성 20~30%)
  • 안빌검사(Anvil test): 종아리 30° 들어올린 후 발바닥에서 축성 압박 → 엉덩관절 통증 유발 시 넓적다리뼈목골절 의심
다리골절 — 무릎·발목·발
  • 정강뼈용기골절: = 범퍼골절(Bumper) = 웬더골절(Fender)
  • 무릎뼈골절: ★드레이어징후검사(Drayer's sign test) / 능동적 폄 가능하면 지지띠 안전 의미
  • 복사골절: 안쪽복사골절은 내번력 작용 시 세모인대가 당겨 발생 / 치료: 보행 석고 3~4주
  • 포트골절(Pott's): 발에 내번력 → 외번(eversion) / 안쪽복사골절 + 세모인대 손상 동반 / 분류: 1도·2도·3도 벌림골절
  • 킨징후검사(Keen's sign test): 발목관절에서 가쪽 5~8cm 위에 부기·피부색 변화 관찰 → 발목관절골절 의심
  • 목말뼈골절(talus fracture): 혈액순환이 매우 취약 / Hawkins 분류 4형 / ★무혈성 괴사: 제2형 50%, 제3형 90%, 제4형 100%
  • 발꿈치뼈골절(calcaneous fracture): 가장 많이 발생하는 발목뼈 골절 / ★척추골절과 동반되는 경우 있음 / 검사: ★호파검사(Hoffa's test)
  • 몰톤검사(Morton's test): 발허리뼈머리 부위 쥐어짬 → 통증 시 발허리뼈머리골절 의심

통증검사 / 말초신경통합성검사 (myotome, dermatome, reflex) / 피부검사 (부기, 발적, 온도 변화) / 균형 및 보행검사
★소리굽쇠검사 (tuning fork test)
방사선 촬영 불가능한 경우 사용 / 골절된 뼈 → 정상 뼈와 다른 약간 둔탁한 소리 / 골절 예상 부위에 진동 주면 뼈끝에서 통증 유발
의학적 검사
  • 병력 청취: 손상 당시 상황, 시기, 심한 정도, 과거 병력
  • 신체검사: 통증·압통·기능장애·변형·비정상적 움직임 + 혈관·신경손상 여부
  • 방사선촬영(X-ray): 가장 중요하고 기본적 / 앞뒤·옆쪽·대각선 촬영
  • CT: 골절 형태 정확히 파악 / 3D-CT: 직접 눈으로 보는 것과 같은 영상
부위별 특수 검사 모음
  • 갭핑검사: ASIS에 힘 가함 → 골반골절 의심
  • 안빌검사: 발바닥 축성 압박 → 넓적다리뼈목골절 의심
  • 드레이어징후검사: SLR 유사 → 무릎뼈골절 의심
  • 킨징후검사: 발목 가쪽 5~8cm 위 부기·압통 → 발목관절골절 의심
  • 호파검사: 발목관절 등쪽굽힘·발바닥굽힘 + 아킬레스힘줄 촉진 → 발꿈치뼈 찢김골절 의심
  • 타진검사: 반사망치로 해부학적 코담배갑 타진 → 손배뼈골절 의심
  • 몰톤검사: 발허리뼈머리 쥐어짬 → 발허리뼈머리골절 의심
X 중재 (Intervention) ▾
1) 의학적 치료
응급 처치
  • 가장 중요: 적절한 부목고정 → 추가적 물렁조직 손상 예방·통증 경감
  • 지방색전증과 쇼크 발생 감소
맞춤 (Reduction)
  • 도수 맞춤: 비수술 / 골절 후 6~12시간 이내 조기에
  • 견인에 의한 맞춤: 뼈견인(skeletal) / 피부견인(skin) / 재전위 위험 없어지면 석고붕대로 전환
  • 개방적 맞춤: 도수 맞춤으로 정확한 정렬 얻기 어려울 때 수술
고정 (Fixation)
  • 외적고정: 석고붕대, 기능성 주형 부목, 외적고정 기구
  • 내적고정: 강선(wire), 핀(pin), 나사와 금속판, 골수내정(intramedullary nail)
  • 보호: 팔걸이(arm sling) / 보행 석고붕대·목발(crutch)
2) 물리치료적 중재 — 고정 시
목표: 손상·초기 염증과 부기 / 점진적 근위축·구축·관절연골 퇴화 / 혈액순환 감소 / 기능적 제한 관리

물리적 인자치료

  • 온습포: 내적고정만 있는 경우 적용 가능 / 타월 4~6겹 / 20~30분
  • 적외선: 내적고정만 시행한 환자 사용 가능 / ★금속이 가까운 경우 화상 주의
  • 극초단파: 깊은 조직 열치료 / ★피부 0.5~2cm 내 금속 삽입물 있으면 사용 금지
  • 초음파: 핏덩이·육아조직 형성 초기에는 사용 금지 / ★금속 내적고정물 있어도 비교적 안심 사용 가능 / 도자를 자주 움직여 열집중 방지
  • NMES: 근수축 유발 / 불용 위축 방지 / 근경축·부기 감소 / 10~20분
  • TENS: 통증 감소 / 관문조절설(gate theory) / 맥동 100~150pps

운동치료

  • ★등척성 운동(isometric exercise): 고정기간 동안 대표적 운동 / 근길이·관절움직임 없는 상태에서 수축 / 불용 위축·근력저하 방지 / 섬유화·강직 예방 / 골절 고정부에 직접 적용 or 인접관절 간접 적용
  • 관리계획: RICE(휴식·냉·압박·올림) / 간헐적 등척성 운동 + 인접부 능동관절운동 / 고정되지 않은 근육군 저항운동
3) 물리치료적 중재 — 고정 제거 후
목표: 운동 시 통증 / 반흔조직 유착으로 인해 ROM·관절놀이 감소 / 근력과 지구력 감소 관리

관리계획

  • 환자 교육: 피해야 할 활동 + 가정 운동 프로그램
  • 다리골절: 부분적 체중부하 / 팔골절: 비스트레스성 활동
  • 관절가동술 (I·II등급 통증조절 / III·IV등급 ROM 증진)
  • 스트레칭운동: ★뼈위축 있으면 재골절 위험 → 초기에 너무 강한 힘 금지
  • 저항운동 진행 순서: 등척성 → 편심성 → 동심성 → 열린 사슬 → 닫힌 사슬
  • ★고정기간 이후 첫 2주는 뼈의 내성이 약하므로 저항운동 조심스럽게 적용
  • 심폐지구력: 뼈유합 완료 전까지 비스트레스성 운동 (유산소운동)

정형도수물리치료

  • 지속적 수동운동(CPM): 수술 직후(최소 3일 이전)부터 / 20~30° 작은 범위 시작 → 매일 10~15°씩 증가 / 운동주기 1cycle/45sec~1cycle/2min / 최소 7일 or 운동범위 개선될 때까지
  • 관절가동술: 골절 부위에 부하 없이 관절놀이(joint play) 회복 / 안정위치(resting position)에서 신연·미끄러짐 이용
  • 물렁조직가동술: 오랜 고정 → 근육 단축 → 유착 형성 → 탄력성 소실 → 통증유발점·관련통 / 혈액순환 증진·유착 제거 → 통증 감소·가동성 증진
  • 수중운동: 부력·정수압·점성·표면장력 이용 / 스트레칭·근력·지구력·균형·보행훈련
