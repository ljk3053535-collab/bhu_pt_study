01
기본 구성 — 뼈와 관절 4개
▼

팔꿈치-아래팔 복합체는 위팔뼈·노뼈·자뼈 3개의 뼈와 4개의 관절로 구성됩니다. 위팔자·위팔노관절은 굽힘/폄을, 몸쪽·먼쪽노자관절은 뒤침/엎침을 담당합니다.

02
골학 — 위팔뼈·자뼈·노뼈 주요 구조물
▼

위팔뼈: 앞면 — 도르래(자뼈관절), 작은머리(노뼈관절), 노오목, 갈고리오목. 뒤면 — 팔꿈치오목. 안쪽위관절융기(Golf's elbow), 가쪽위관절융기(Tennis elbow).

자뼈: 팔꿈치머리돌기(세갈래근 부착), 갈고리돌기, 도르래패임, 자뼈거친면(→위팔근 부착).

노뼈: 노뼈머리(관절연골 280° 덮임), 노뼈거친면(→위팔두갈래근 부착), 자패임(먼쪽노자관절).

핵심 암기
자뼈거친면→위팔근 / 노뼈거친면→위팔두갈래근
03
관절학 — 박굽이 수치, 관절 특성
▼

팔꿈치 = 수정된 경첩관절. 위팔자관절이 주 안정성, 위팔노관절이 압박력 전달 + 박굽이 저항 50%.

정상 박굽이각도: 15° (여성 2° 더 큼, 평균 13°). 과도 박굽이: 30°. 안굽이: 5° 미만.

04
관절주위 결합조직 — MCL·LCL·Terrible Triad
▼

MCL 앞섬유 ★: 가장 강함, 박굽이 주 저항, 박굽이+폄 시 장력 최대. Tommy John surgery = 자가건 이식.

LUCL ★: 가쪽(자쪽)결인대, 뒤-가쪽 회전 불안정성 방지.

Terrible Triad: 팔꿈치 탈구 + 노뼈머리 골절 + 갈고리돌기 골절 → 인공노뼈머리 삽입.

05
팔꿈치 운동형상학 — ROM·관절면 운동
▼

전체 ROM: -5°~145°. 기능적 ROM: 30°~130° (100°). 최대 굽힘 토크 각도: 85~95°.

위팔자관절: 오목↔볼록, 굽힘 시 구르기·미끄러짐 같은 방향.
위팔노관절: 볼록↔오목, 능동 굽힘 시 작은머리 쪽으로 당겨짐.

06
뼈사이막 — 중심띠, 압박력 전달
▼

중심띠(Central band): 주된 성분, 두께 2배, 무릎힘줄과 유사한 장력.

손목 압박력: 노손목관절 80% → 노뼈 → 중심띠 → 자뼈 / 자손목공간 20% → 자뼈 직접.

완전 엎침: 최소 팽팽 / 완전 뒤침: 최대 팽팽.

07
아래팔 관절 — 노자관절, TFCC, 당겨진팔꿈치
▼

몸쪽노자관절: 섬유뼈고리 = 머리띠인대 75% + 자뼈 노패임 25%. 네모인대: 몸쪽노자관절 안정 (뒤침 시 긴장).

당겨진팔꿈치증후군: 어린이, 손 강하게 당김 → 노뼈머리 탈구. 정복 = 뒤침-굽힘.

먼쪽노자관절: 정상 노뼈 이동 5.5mm. TFCC = 먼쪽노자관절 일차 안정자. TFCC 마모 = RA 첫 번째 임상 징후.

08
돌림 ROM — 뒤침/엎침 가동범위
▼

뒤침: 85°, 엎침: 75°. 기능적 각 50° (총 100°). 중립위 = thumb-up position.

뒤침 제한 구조: 원엎침근, 네모엎침근, 노쪽손목굽힘근, TFCC(바닥쪽), 뼈사이막 중심띠, 네모인대.

엎침 제한 구조: 위팔두갈래근, 뒤침근, 노쪽 손목 폄근들, TFCC(등쪽).

CKC vs OKC: 먼쪽노자관절 구르기·미끄러짐 방향이 CKC=반대, OKC=같음.

09
팔꿈치 굽힘근육 — 4가지, 생체역학
▼

위팔두갈래근: 노뼈거친면+섬유띠 / 굽힘+뒤침 동시 최대 EMG / 엎침 시 활성 감소.

위팔근 ★사역마: 자뼈거친면 / 굽힘 전용 / 용적 59.3, 단면적 7.0cm² 최대 / 뒤침·엎침 무관하게 항상 동원.

위팔노근: 가장 긴 근육 / 내적 모멘트팔 5.19cm 최대 / 활시위 형성.

원엎침근: 굽힘+엎침 / 최소 단면적 3.4cm².

굽힘>폄 약 70% / 최대 굽힘 토크 85~95° / 뒤침 시 굽힘 토크 20~25% 증가.

10
팔꿈치 폄근육 — 세갈래근, 팔꿈치근
▼

위팔세갈래근 안쪽갈래 ★: 폄근의 사역마, 가장 먼저 동원. 긴갈래: 최대 용적(66.6), 예비적 폄근. 가쪽갈래: 중간 파워 시.

팔꿈치근: 느린연축, 바탕적 안정, 폄 토크 15%.

최대 폄 토크 각도: 80~90°.

절약의 법칙: 낮은 수준(팔꿈치근+안쪽갈래) → 중간(가쪽갈래 추가) → 높은(긴갈래 예비 동원).

C⁶ 팔다리마비: 큰가슴근 빗장갈래+앞어깨세모근 → 어깨 굽힘으로 팔꿈치 잠금.

11
뒤침·엎침 근육 — 기능, 절약의 법칙
▼

뒤침근: 낮은 파워 / 노신경 깊은가지 사이 통과 (→뒤사이신경).

위팔두갈래근: 뒤침근 단면적 3배 / 90° 굽힘+빠른 뒤침 최대.

원엎침근: 정중신경이 위팔갈래·자갈래 사이 통과 (→앞뼈사이신경). 신경압박 위험.

네모엎침근: 항상 먼저 동원 / 먼쪽노자관절 안정화.

뒤침 토크 > 엎침 토크 약 25% / 이유: 뒤침근 단면적 ≈ 엎침근의 2배.

12
신경지배 — 주요 신경과 위험부위
▼

위팔근·위팔두갈래근·위팔노근: C⁵C⁶. 원엎침근: C⁶C⁷. 위팔세갈래근·팔꿈치근: C⁷C⁸. 네모엎침근: C⁸T¹.

노신경 위험부위: 뒤침근의 얕은·깊은 섬유 사이 → 뒤사이신경.

정중신경 위험부위: 원엎침근의 두 갈래 사이 → 앞뼈사이신경.
