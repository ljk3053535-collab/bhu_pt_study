03

손의 활 (Arches) & 운동형학

몸쪽가로활

Proximal transverse arch

  • 손목뼈 먼쪽줄에 의해 형성
  • 손굴(carpal tunnel) 형성
  • 쐐기돌 = 알머리뼈(capitate)
  • 단단하고 고정적
먼쪽가로활

Distal transverse arch

  • MCP관절 사이로 지나감
  • 양 옆면은 가동성 있음
  • 손 접기 동작 가능
  • 쐐기돌 = 둘째·셋째 MCP
세로활

Longitudinal arch

  • 둘째·셋째 손가락선 형성
  • CMC관절로 손목뼈에 연결
  • 먼쪽 끝 = 매우 가동적
  • 세로 안정성 제공
📌 3개의 모든 활은 역학적으로 서로 연결. 쐐기돌은 둘째·셋째 MCP관절에 의해 제공되며 세로활과 먼쪽가로활 모두에 작용.
기능적 위치 (Position of Function) ★★★
손목: 20~30° 폄 + 약간 자쪽치우침 MCP: 35~45° 굽힘 PIP·DIP: 15~30° 굽힘 엄지 CMC: 35~45° 벌림 → 손바닥을 약간 개방해서 오므린 위치 → 잡기 기능 최대화 / 근육 최적 길이 유지 → 임상: 뇌졸중·척수손상 환자에게 보조기 적용 시 이 위치 유지
04

손목손허리관절 (CMC Joint) ★★★

CMC 관절별 가동성
CMC관절하는 뼈가동성역할
2번째작은마름뼈 + 알머리뼈 + 큰마름뼈 일부가장 적음중심기둥 (central pillar) — 안정성
3번째알머리뼈 먼쪽 관절면가장 적음
4번째갈고리뼈 먼쪽 + 알머리뼈 일부약 20° 굽힘/폄, 27° 안쪽돌림주변부 — 가동성 / 맞섬 보조
5번째갈고리뼈 먼쪽 관절면만약 44° (4번 자유로울 때)
엄지 CMC관절 — 안장관절 ★★★
  • 큰마름뼈(trapezium) ↔ 첫째손허리뼈
  • 안장관절(saddle joint) — 각 면이 한쪽 볼록+다른 쪽 오목
  • 가장 복잡한 CMC관절
  • 완전한 맞섬(opposition) 가능
  • 자유도 2 (굽힘·폄 + 벌림·모음)
엄지 CMC 맞섬 2단계
1단계: 엄지 손허리뼈 벌림 (큰마름뼈 관절면 → 바닥쪽 구르기 + 등쪽 미끄러짐) 2단계: 손허리뼈 → 손바닥 가로질러 새끼손가락 쪽으로 굽힘 + 안쪽돌림 → 완전 맞섬 = CMC 닫힌위치 → 45~60도 안쪽돌림 포함
엄지 CMC 인대 (표 8.1) ★
인대몸쪽 부착먼쪽 부착특징 & 팽팽해지는 때
앞(바닥쪽)빗인대큰마름뼈 바닥쪽결절엄지 손허리뼈바닥 바닥쪽("부리")가늘고 약함 / 맞섬·굽힘·벌림 시 느슨
자쪽결인대가로손목인대 노쪽면엄지 손허리뼈바닥 바닥쪽-자쪽벌림·폄 시 팽팽
손허리뼈사이인대둘째손허리뼈바닥 등쪽-노쪽엄지 손허리뼈바닥 바닥쪽-자쪽맞섬·굽힘·벌림 시 팽팽
노쪽결인대†큰마름뼈 노쪽면엄지 손허리뼈바닥 등쪽두껍고 강함 / 감각섬유 풍부 / 맞섬된 CMC 일차 안정자
뒤빗인대†큰마름뼈 뒤-노쪽 모서리엄지 손허리뼈바닥 바닥쪽-자쪽두껍고 강함 / 감각섬유 풍부 / 맞섬된 CMC 일차 안정자
🔴 임상: CMC 골관절염(50~60대 여성 호발) → 앞빗인대·노쪽결인대 파열 → 노쪽 탈구 + 혹(hump) 형성. 치료: NSAID·부목·코티코스테로이드 → 실패 시 관절성형술.
05

손허리손가락관절 (MCP Joint) ★★★

일반 특징
  • 볼록한 손허리뼈머리 ↔ 오목한 몸쪽손가락뼈
  • 비교적 큰 달걀형 관절
  • 자유도 2: 굽힘·폄(시상면) + 벌림·모음(이마면)
  • 손허리뼈머리 캠(cam) 형태 → 굽힘 시 결인대 팽팽
손가락굽힘 범위
집게손가락(2번)약 90°
새끼손가락(5번)약 110~115°
벌림·모음셋째 기준 양쪽 약 20°
MCP 결합조직 구조
MCP관절 안정화 구조 ├── 노쪽결인대 + 자쪽결인대 │ ├── 끈부분(cord part): 굵고 강함 │ └── 덧부분(accessory part): 바닥판에 부착 ├── 바닥판(palmar plate) │ └── 굳고 뻣뻣 / 과다폄 제한 │ └── 굽힘건 도르래(A₁ pulley) 역할 ├── 섬유손가락집 │ └── 바닥판 바닥쪽(앞쪽)면에 고정 └── 깊은가로손허리인대 (3개) └── 2~5번째 손허리뼈 바닥판 연결
📌 MCP 닫힌위치 = 굽힘 약 60~70° (캠 형태 때문에 굽힘 시 결인대 끈부분 신장으로 일치성 최대). 임상: 골절 후 부목 고정 시 MCP를 70° 굽힘 상태로 → 폄구축 방지!
📌 반대로 IP관절(PIP·DIP)의 닫힌위치 = 완전 폄 → 고정 시 펴진 위치로 (굽힘구축 방지)

손가락뼈사이관절 (IP Joint)

항목PIP관절DIP관절
위치몸쪽손가락뼈머리 ↔ 중간손가락뼈바닥중간손가락뼈머리 ↔ 먼쪽손가락뼈바닥
자유도1 (굽힘·폄만)
굽힘 범위약 100~120°약 70~90°
폄 범위과다폄 약간 허용중립 넘어 약 30° 허용
관절면 형태은촉이음(tongue-in-groove) — 두 개의 둥근 관절융기 ↔ 두 개의 알은 오목 관절면
특징 구조제지고삐인대(check-rein lig.) 있음
→ 병리조직화 시 굽힘구축 유발		제지고삐인대 없음
빗지지띠인대 (ORL) — PIP↔DIP 협응 ★
빗지지띠인대(oblique retinacular ligament, ORL) = PIP 바닥쪽 섬유손가락집 → DIP 등쪽 가쪽띠 (사선 주행) 기전: ① 내재성 근육 활성 → PIP 폄 ② PIP 폄 → ORL 신장 ③ 신장된 ORL → 수동력 DIP쪽 전달 ④ DIP 수동 폄 유발 → "연결인대(link ligament)" = PIP·DIP 폄이 협응되는 기전 임상: ORL 타이트 → PIP 굽힘구축 + DIP 과다굽힘 → 뒤퓨트랑구축(Dupuytren's contracture) 연관

▸ 운동지배

외재성 굽힘근 & 굽힘근도르래 ★★★

근육부착주작용특징
알은손가락굽힘근
FDS		 중간손가락뼈 바닥쪽 양 옆면 PIP 굽힘 PIP 수준에서 갈라져 FDP 힘줄 통과시킴 (분리)
깊은손가락굽힘근
FDP		 먼쪽손가락뼈 바닥쪽면 DIP 굽힘 (유일) 집게손가락은 독립조절 가능 / 자쪽윤활집에 쌓임
긴엄지굽힘근
FPL		 엄지 먼쪽손가락뼈 바닥쪽 엄지 IP·MCP·CMC 굽힘 노쪽윤활집에 둘러싸임 / CMC에서 상당한 굽힘 토크
굽힘근도르래 시스템
섬유손가락집 안에 존재 ├── 고리도르래(annular pulley): A₁~A₅ │ ├── A₂: 몸쪽손가락뼈몸통 (가장 중요) │ └── A₄: 중간손가락뼈몸통 (두 번째로 중요) └── 십자도르래(cruciate pulley): C₁~C₃ 엄지: 고리인대(A₁·A₂) + 빗인대(oblique lig.)
임상 — 활시위·방아쇠손가락
⚠️
활시위(bowstringing): 도르래 손상 → 힘줄이 관절 돌림축에서 멀어짐 → 굽힘 효율 급감

방아쇠손가락(trigger finger): 힘줄윤활막염 → 결절 형성 → A₁ 도르래 협착 → 보존치료(부목·코티손) → 실패 시 A₁ 도르래 수술적 절개
🏥
손목굴증후군(CTS): 정중신경 + 굽힘근 힘줄 9개가 손목굴 통과 → 압박 → 감각이상·엄지두덩 위축. 치료: 부목·교육 → 수술적 감압

효과기로서의 손 — 잡기(Prehension) 분류 ★★★

01
강력한 잡기
Power grip
큰 힘, 정밀함 불필요. 공 모양·원통 모양 물체. 손가락굽힘근+내재성+엄지 모음·굽힘근. 예: 망치
02
정밀한 잡기
Precision grip
조절 필요, 엄지 부분 벌림+나머지 굽힘. 3점 집기 = 집게·가운뎃+엄지 접촉. 예: 야구공·계란
03
강력한(열쇠) 집기
Key/Lateral pinch
엄지↔집게 가쪽모서리 사이 큰 힘. 엄지모음근+첫째등쪽뼈사이근 필요. 예: 열쇠
04
정밀한 집기
Precision pinch
엄지↔집게 미세조절, 힘 불필요. 손끝집기(tip-to-tip), 손가락패드 집기 포함. 예: 바늘집기
05
갈고리식 잡기
Hook grip
엄지 미사용. PIP·DIP 부분 굽힘. 장시간 정적 유지에 사용. 예: 가방 들기

류마티스관절염(RA) 손 변형 ★★★

엄지의 지그재그변형
CMC관절 굽힘·모음 → 손허리뼈 노쪽 탈구 → MCP관절 과다폄 → IP관절 굽힘 = CMC(굽힘+모음) + MCP(과다폄) + IP(굽힘) 지그재그
MCP관절 변형 2가지
바닥쪽탈구 (Palmar dislocation)
  • 굽힘근 활시위 힘 → 결인대 파열
  • 몸쪽손가락뼈 바닥쪽 탈구
  • 손의 세로활·가로활 붕괴
  • → 손 편평해짐
자쪽이동 (Ulnar drift)
  • 자쪽 방향 활시위 힘
  • 등쪽덮개 가로섬유 파열
  • ED힘줄 자쪽으로 이동 → 굽힘 토크 발생
  • → 악순환: 자쪽이동↑ → 모멘트팔↑ → 더 큰 토크 → 완전 탈구
손가락 RA 변형 2가지 ★★★
🦢 백조목변형 (Swan-neck)
PIP 과다폄 + DIP 굽힘
  • 바닥판 과신장·약화
  • 내재성 근육 수축·섬유증
  • 가쪽띠 등쪽으로 이동
치료: PIP 과다폄 차단 보장구 / 바닥판 복구 또는 전관절성형술
VS
🔘 단추구멍변형 (Boutonnière)
PIP 굽힘 + DIP 과다폄
  • 중심띠 + 세모인대 파열
  • 가쪽띠 바닥쪽으로 미끄러짐
  • PIP 굽힘시키는 힘으로 전환
치료: PIP 폄 상태 부목 / 중심띠 복구 또는 가쪽띠 재정렬 수술
