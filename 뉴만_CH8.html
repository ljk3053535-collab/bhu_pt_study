손가락뼈
종류특이사항
총 14개엄지 2개, 나머지 각 3개
먼쪽손가락뼈거친면(tuberosity) — 속질 고정
몸쪽가로활
  • 손목뼈 먼쪽줄 형성
  • 손굴(carpal tunnel) 형성
  • 쐐기돌 = 알머리뼈
  • 고정적, 단단
먼쪽가로활
  • MCP관절 사이
  • 쐐기돌 = 2·3번 MCP
  • 양 옆면 가동성 있음
세로활
  • 2·3번 손가락선 형성
  • CMC로 손목뼈 연결
  • 먼쪽 끝 매우 가동적
기능적 위치 ★★★
손목: 20~30° 폄 + 약간 자쪽치우침 MCP: 35~45° 굽힘 PIP·DIP: 15~30° 굽힘 엄지 CMC: 35~45° 벌림 → 임상: 뇌졸중·척수손상 환자 보조기 적용 시 이 위치 유지
CMC가동성역할
2·3번째가장 적음중심기둥 — 안정성
4번째약 20° 굽힘/폄, 27° 안쪽돌림주변부 — 가동성
5번째약 44°주변부 — 맞섬 보조
엄지 CMC — 안장관절 (Saddle Joint)
맞섬 2단계: 1단계: 손허리뼈 벌림 (바닥쪽 구르기 + 등쪽 미끄러짐) 2단계: 손바닥 가로질러 굽힘 + 안쪽돌림 → 완전 맞섬 = CMC 닫힌위치 (45~60° 안쪽돌림)
🔴 CMC 골관절염(50~60대 여성 호발) → 앞빗인대·노쪽결인대 파열 → 노쪽 탈구 + 혹(hump) 형성. 치료: NSAID → 관절성형술
📌 강한 인대 2개: 노쪽결인대 + 뒤빗인대 — 두껍고 강함, 감각섬유 풍부, 맞섬된 CMC 일차 안정자
📌 MCP 닫힌위치 = 60~70° 굽힘 (캠 형태 → 결인대 팽팽). 골절 후 부목: MCP 70° 굽힘 → 폄구축 방지!
반대로 IP관절 닫힌위치 = 완전 폄 → 굽힘구축 방지
항목PIPDIP
굽힘 범위100~120°70~90°
특징 구조제지고삐인대 있음 → 굽힘구축 유발제지고삐인대 없음
빗지지띠인대 (ORL) ★
PIP 바닥쪽 → DIP 등쪽 가쪽띠 (사선 주행) 내재성 근육 활성 → PIP 폄 → ORL 신장 → DIP 수동 폄 → "연결인대": PIP·DIP 폄이 협응되는 기전 임상: ORL 타이트 → PIP 굽힘구축 + DIP 과다굽힘
노신경Radial
운동: 외재성 펴근육 전체 (ED, EI, EDM, EPL, EPB, APL)
감각: 손·손목 등쪽면, 물갈퀴공간 등쪽
정중신경Median
운동: FDS, FDP 가쪽절반, FPL + 엄지두덩 3근육 + 가쪽 2개 벌레근
감각: 엄지~셋째손가락 + 넷째 가쪽 1/2 바닥쪽
⚠ 손상 → 엄지두덩 위축, 맞섬 불가 (원숭이손)
자신경Ulnar
운동: FDP 안쪽절반, 새끼두덩 전체, 안쪽2 벌레근, 모든 뼈사이근, 엄지모음근
감각: 새끼손가락 전체 + 넷째 자쪽 1/2
⚠ 손상 → 안쪽 내재성 마비 → 갈퀴(claw)
📌 척수근: C⁶ = 엄지·집게 / C⁷ = 가운뎃손가락 / C⁸ = 반지·새끼 | 내재성 신경지배: 축삭 1개 → 100개 근섬유 (정밀 조절!)
근육부착주작용특징
FDS중간손가락뼈 양 옆면PIP 굽힘PIP에서 갈라져 FDP 통과시킴
FDP먼쪽손가락뼈 바닥쪽DIP 굽힘 (유일)집게손가락 독립조절 가능
FPL엄지 먼쪽손가락뼈 바닥쪽엄지 IP·MCP·CMC 굽힘노쪽윤활집에 둘러싸임
굽힘근도르래 시스템
고리도르래(Annular): A₁~A₅ A₂: 몸쪽손가락뼈몸통 (가장 중요) A₄: 중간손가락뼈몸통 (두 번째) 십자도르래(Cruciate): C₁~C₃ 엄지: A₁·A₂ + 빗인대
⚠️ 방아쇠손가락: 힘줄윤활막염 → 결절 → A₁ 도르래 협착 → 보존치료(부목·코티손) → 실패 시 A₁ 도르래 절개
🏥 CTS: 정중신경 + 굽힘근 힘줄 9개 압박 → 감각이상·엄지두덩 위축
힘줄고정작용 (Tenodesis)
손목 폄 → 손가락 수동 굽힘 손목 굽힘 → 손가락 수동 폄 C⁶ SCI 활용: 짧은노쪽손목폄근(유지) → 능동 손목 폄 → 굽힘근 수동 신장 → 컵 잡기 가능!

손 펴기 3단계 ★★★
A. 초기: ED가 MCP 일차 폄 / 내재성 근육 사용 X B. 중간: 내재성 근육 개입 → PIP·DIP 폄 시작 / ED: MCP 과다폄 못 하도록 굽힘 토크 C. 후기: 완전한 손가락 폄 완성 / ED + 내재성 협력

01
강력한 잡기
Power grip
큰 힘, 정밀 불필요. 손가락굽힘근+내재성+엄지 모음·굽힘. 예: 망치
02
정밀한 잡기
Precision grip
조절 필요, 엄지 부분 벌림+나머지 굽힘. 3점 집기. 예: 야구공·계란
03
강력한 집기
Key/Lateral pinch
엄지↔집게 가쪽 큰 힘. 엄지모음근+첫째등쪽뼈사이근. 예: 열쇠
04
정밀한 집기
Precision pinch
엄지↔집게 미세조절. 손끝집기(tip-to-tip). 예: 바늘집기
05
갈고리식 잡기
Hook grip
엄지 미사용. PIP·DIP 부분 굽힘. 정적 유지. 예: 가방 들기
엄지 지그재그변형
CMC (굽힘+모음) → 손허리뼈 노쪽 탈구 → MCP 과다폄 → IP 굽힘 = 지그재그
손가락 RA 변형 2가지 ★★★
🦢 백조목변형 (Swan-neck)
PIP 과다폄+DIP 굽힘
  • 바닥판 과신장·약화
  • 내재성 근육 수축·섬유증
  • 가쪽띠 등쪽으로 이동
치료: PIP 과다폄 차단 보장구
VS
🔘 단추구멍변형 (Boutonnière)
PIP 굽힘+DIP 과다폄
  • 중심띠 + 세모인대 파열
  • 가쪽띠 바닥쪽으로 미끄러짐
  • PIP 굽힘시키는 힘으로 전환
치료: PIP 폄 상태 부목
바닥쪽탈구
  • 굽힘근 활시위 힘 → 결인대 파열
  • 손 세로활·가로활 붕괴 → 편평해짐
자쪽이동 (Ulnar drift)
  • 등쪽덮개 가로섬유 파열
  • ED힘줄 자쪽 이동 → 굽힘 토크 발생
  • 악순환: 자쪽이동↑ → 완전 탈구
병변신경/구조임상 소견치료
정중신경 손상C⁶~T¹엄지두덩 위축, 맞섬 불가, 원숭이손, 가쪽 3.5지 감각이상감압/봉합, 엄지두덩 재건
자신경 손상C⁸~T¹갈퀴변형(반지·새끼), 새끼두덩 위축, 자쪽 1.5지 감각이상봉합, MCP 굽힘 보조 보장구
CTS정중신경 압박야간 통증·감각이상, 엄지두덩 위축, Tinel·Phalen (+)부목·NSAID → 수술적 감압
방아쇠손가락A₁ 도르래 협착잠김, 찰각 소리, 결절 촉지부목·코티손 → A₁ 도르래 절개
CMC 골관절염엄지 CMC50~60대 여성, 혹 형성, 노쪽 탈구, 집기 통증부목·NSAID → 관절성형술
C⁶ 척수손상외재성 굽힘근 마비손가락 굽힘 불가, 손목 폄 유지힘줄고정(tenodesis) 재활
