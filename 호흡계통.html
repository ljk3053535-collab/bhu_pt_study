👃 1. 코 (Nose)
  • 바깥코: 얼굴에 보이는 코 / 뼈(위쪽)+연골로 뼈대 / 표면=피부, 속=점막
  • 코안: 코중격으로 좌우 분리 / 천장=이마뼈·벌집뼈·나비뼈 / 바닥=입천장
  • 코선반 3개: 위·중간·아래코선반 → 선반 사이=위·중간·아래콧길
  • 후각부위: 후각세포 분포 / 호흡부위: 혈관·샘 풍부

🗣️ 2~3. 인두 & 후두
  • 머리뼈 바닥 → 제6목뼈 높이까지 뻗어 있음
  • 분류: 코인두(공기만) / 입인두(공기+음식) / 후두인두(공기+음식)
  • 코인두: 호흡계통에만 속함 / 입·후두인두: 호흡+소화 모두

  • 후두안뜰(Vestibule): 성대문 위 공간 / 안뜰주름(거짓성대) - 기도보호역할
  • 후두실(Ventricle): 참성대·거짓성대 사이 함몰 공간
  • 성대주름(참성대): 성대근+성대인대+성대점막으로 구성
  • 성대문틈새(Rima glottidis): 좌우 성대주름 사이, 숨쉴 때 열림·발성 시 닫힘
  • 반지방패근: 성대를 팽팽하게 당겨 고음 생성
  • 뒤반지모뿔근: 유일하게 성대문을 여는 근육(벌림근)
  • 탄력원뿔: 반지연골~성대 연결 원뿔형 막 / 그 윗부분 = 성대인대
🌿 4~5. 기관 & 기관지
  • 길이: 약 10cm (제6~7목뼈 ~ 제5등뼈)
  • 구조: 앞-C자형 연골 / 뒤-민무늬근육+인대성막
  • 점막: 섬모원기둥상피 + 술잔세포
  • 기관갈림: 제5등뼈 수준 → 좌우 주기관지 분기 (우24°, 좌46°)

🫁 6. 허파 (Lung)
  • 오른허파: 620g / 3엽(위·중간·아래) / 수평틈새(위-아래) + 빗틈새(위엽→위·중간)
  • 왼허파: 570g / 2엽(위·아래) / 빗틈새 / 허파혀·심장패임
  • 실질혈관: 기관지동맥+기관지정맥 → 허파 자체 영양공급
  • 기능혈관: 허파동맥+허파정맥 → 가스교환
세포면적기능
I형 세포95%가스교환 부위 / 혈액-공기 장벽 형성
II형 세포5%표면활성제(surfactant) 분비 → 표면장력↓, 허파꽈리 허탈 방지 / 손상 시 I형·II형으로 분화
허파큰포식세포—단핵구 유래 / 먼지·세균·이물질 포식 / 허파내 무균 유지
🏛️ 7. 가슴 구조물
  • 가슴막강: 음압 형성 / 가슴막액이 차 있어 마찰 감소
  • 위세로칸: 대동맥궁, 기관, 가슴샘
  • 앞세로칸: 지방, 가슴샘
  • 중간세로칸: 심장, 심장막, 허파동맥·정맥
  • 뒤세로칸: 식도, 내림대동맥, 교감신경줄기, 가슴림프관

📈 산소해리곡선
이동방향조건의미
왼쪽 이동PCO₂↓, pH↑, 온도↓, 2,3-DPG↓O₂ 결합력↑ (헤모글로빈이 O₂를 잘 놓지 않음)
오른쪽 이동PCO₂↑, pH↓, 온도↑, 2,3-DPG↑O₂ 해리력↑ (조직에 O₂를 잘 줌)
  • 산소분압 높은 곳(100mmHg): 포화도 90%, 해리도 10%
  • 산소분압 낮은 곳(30mmHg): 포화도 40%, 해리도 60%
  • 포화도 + 해리도 = 항상 100%
