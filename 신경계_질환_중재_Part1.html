💥II. 외상성 뇌손상
1. 정의 & 분류
외부 물리적 힘에 의한 뇌손상. 낙상·교통사고·스포츠 손상 등이 주요 원인

일차적 뇌손상

  • 타격손상(coup): 타격 가해진 쪽 손상 (가속손상)
  • 반대편타격(countercoup): 반대쪽 손상 (감속손상)

이차적 뇌손상

  • 뇌부종, 뇌내출혈
  • 허혈성 뇌손상
  • 뇌이탈, 뇌감염
2. 란초로스 아미고스 10단계
단계분류반응
I낮은수준반응 없음
II일반적 반응 (목적 없음)
III국소적 반응 (구체적이나 지속적이지 않음)
IV중간수준혼란-흥분(agitated)
V혼란-부적합 (단순지시 반응)
VI혼란-적합 (목표지향적이나 지시 필요)
VII높은수준자동-적합 (판단·문제해결 부족)
VIII목적-적합 (미세한 장애)
IX목적-적합 (전반적 암시 필요)
X목적-적합 (임시적 차단 필요)
3. 글래스고 혼수척도 (GCS)
영역최대 점수
눈뜨기 반응 (E)4점
언어 반응 (V)5점
운동 반응 (M)6점
총 15점 — 낮을수록 신경학적 상태 나쁨
🏥 임상포인트 — 외상성 뇌손상

• GOS 1등급: 사망 / 5등급: 양호한 회복
• GOAT: 10개 항목 100점, 낮을수록 심한 기억상실
• 딴곳뼈되기: 연부조직에 성숙 골층판뼈 생성 → 엉덩·무릎관절 호발
• DVT: 약 40~54% 발생, 폐색전증 80%

⚙️III-1. 파킨슨병
1. 정의 & 특징
도파민 경로 장애 + 바닥핵 손상으로 발생하는 만성 진행성 퇴행성 신경질환
2. 4대 주요 운동 증상
증상세부 내용
안정시 떨림수의운동 시 사라짐 / 알약돌리기운동 / 손목손허리관절
경축(rigidity)속도 무관 일정 저항 / 톱니바퀴형 vs 납파이프형
운동못함증/완만증운동 시작 어려움(akinesia) / 동결 현상(freezing)
자세 불안정성고릴라자세 / 몸통 굽힘근 > 폄근 약화
3. 보행 특징
  • 종종걸음 (shuffle gait): 짧은 스텝, 발 질질 끌기
  • 가속보행: 점점 빨라지는 보행
  • 팔 흔들기 감소, 회전 어려움
  • 작은글씨증, 가면얼굴, 단조로운 언어
4. 평가도구
도구내용
mHY 척도진행 상황 8단계
mSEADLSADL 능력 (100%=완전독립, 0%=기능 없음)
PDQ-39삶의 질, 8개 항목 39개 설문
UPDRS임상연구 가장 일반적, 4개 파트
🏥 임상포인트 — 파킨슨병

• 안정시 떨림은 수의운동 시 사라짐 ↔ 소뇌 활동떨림은 목표 접근 시 심해짐
• 톱니바퀴 = 불규칙적 저항 + 교대수축이완
• 납파이프 = 균일하고 부드러운 저항

🧒IV. 뇌성마비
1. 정의 & 발생 원인
뇌 성숙 이전 다양한 원인으로 비정상적 근긴장도 발생 → 운동·자세 장애를 보이는 비진행성 증후군
원인비율예시
출생 전30%유전, 기형, 모체감염
출생 시60%조산, 핵황달, 산소결핍
출생 후10%뇌출혈, 수막염, 중독
2. 유형별 특징

실조형2~5%소뇌균형·협응 부족, 넓은 기저면
저긴장형--개구리다리자세, 근긴장도 매우 낮음
3. 운동발달 이정표

🏥 임상포인트 — 뇌성마비

• 강직형 자세: thumb in palm, scissoring posture
• 저긴장형: 개구리다리 자세 (엉덩관절 굽힘-벌림-가쪽돌림)
• 발생빈도: 남 1.4 : 여 1
• 강직형은 조산 증가로 발생 증가 / 무정위운동형은 질식 감소로 발생 감소

🏥 임상포인트 — 척수손상

• 호발연령: 20~40세 80% / 남:여 = 2.5:1
• 욕창 호발: 엉치, 발뒤꿈치, 넙다리큰돌기, 종아리뼈머리, ASIS
• DVT: 손상 후 2개월 이내 호발

⚡신경계 중재 I — 뇌졸중
1. 급성기 치료 (발생 72시간 이내)
  • 욕창방지: 2시간마다 자세 변경
  • 마비쪽이 밝은 곳으로 향하게 (인식 위해)
  • 바로누운자세: 마비쪽 어깨관절 가쪽돌림·벌림 30°
  • 기립성저혈압 방지: 침대 각도 서서히 증가
2. 보행훈련 단계
상황순서
평지지팡이 → 마비쪽 발 → 비마비쪽 발
계단 오르기비마비쪽 발 → 지팡이 → 마비쪽 발
계단 내려가기지팡이 → 마비쪽 발 → 비마비쪽 발
3. 물리적 인자치료
치료목적
핫팩만성기 근긴장도 감소, 통증 경감
냉치료강직 억제, 통증 조절
FES(기능적 전기자극)어깨 불완전탈구, 보행 보조
TENS통증 완화 (비침습적)
🏥 임상포인트 — 뇌졸중 중재

• 지팡이 높이: 넙다리뼈 큰돌기에 위치
• 팔꿉관절 약 20~30° 굽힘되도록 비마비쪽에 잡음
• 매트훈련 구르기: 비마비쪽 손으로 마비쪽 손 잡고 굽힘 → 머리부터 분절 돌림
