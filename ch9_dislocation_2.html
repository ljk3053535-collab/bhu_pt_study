🔴 접시위팔관절 탈구 합병증

힐-세치병변 (Hill-Sachs)

앞쪽 탈구 → 위팔뼈 머리 뒤가쪽 골결손(함몰). 첫 탈구의 35~40% 발생.

반가르트·슬랩병변

접시오목 끝 관절주머니·접시테두리 단열/견열. 만성 불안정성. 40대↑ 회전근개 동반 손상.

겨드랑신경 병변

앞쪽 탈구에서 多. 어깨세모근 마비 + 팔 위가쪽 감각 상실. 압박(neuropraxia) → 몇 주 회복, 축삭절단(axonotmesis) → 더 오래 걸림.

동반 골절 & 습관성

위팔뼈 큰결절 동반 골절 가능. 관절면·접시오목 손상 심할 때 습관성 탈구 유발.

🦴 1. 접시위팔관절 탈구 (Glenohumeral Joint)

개요: 접시위팔관절은 접촉면이 위팔뼈 머리의 1/3 정도 → 넓은 ROM 허용, 매우 불안정. 신체관절 중 재발성 탈구 가장 빈발. 동적 안정성: 위팔두갈래근힘줄, 회전근개 (특히 접시테두리에 섞여 동적 인대 역할).

앞쪽 탈구 유형특징
부리돌기밑 (가장 흔함)벌림·폄·가쪽돌림 → 앞·아래 이동. 봉우리 돌출, 어깨 편평, 위팔뼈머리 돌출, 팔 약간 벌림·팔꿈치 굽힘 지지
접시오목밑노인, 과도한 벌림 → 봉우리 지렛대 → 신경혈관 손상 흔함
빗장밑위팔뼈머리가 빗장 아래로 전위, 매우 드뭄
뒤쪽 탈구전체의 2%, 간질·전기 쇼크. 모음·굽힘·안쪽돌림
합병증내용
힐-세치병변앞쪽 탈구 결과 → 위팔뼈머리 뒤가쪽 골결손(함몰). 첫 탈구 35~40% 발생. 반복 손상 시 잔여병변.
반가르트병변접시오목 끝 관절주머니 앞쪽 + 접시테두리 단열/견열
슬랩병변(SLAP)접시오목 끝 접시테두리 떨어짐. 40대↑ 회전근개 잦은 손상
겨드랑신경 병변앞쪽 탈구에 多. 어깨세모근 마비 + 위가쪽 감각 상실

중재 — 정복 방법:

  • 히포크라테스당김법: 치료사 발 겨드랑이, 손목 잡아 가쪽·안쪽돌림하며 당김
  • 스팀슨당김법: 엎드려 팔 늘어뜨림 + 3kg 추 손목 → 약 30분 후 정복
  • 밀치당김법 (가장 부드러움): 벌림·안쪽돌림 자세 유지 → 5~10분 후 벌림 120°+가쪽돌림 60° → 자발 정복

물리치료: 팔걸이 착용 → 팔꿈치관절 등척성 운동 및 팔걸이 제거 후 팔꿈치 펌운동. 회전근개 등척성 운동. 가쪽돌림·안쪽돌림: 노인 2~3주, 젊은 환자 6주 후부터.

⭐ TIP — 위등세모근(UT) 약화·마비

UT 약화 마비 → ACJ 위로 올라감 → 탈구 쉬움. 편마비환자에서 탈구 잘 일어남. 접시오목의 아래쪽 돌림 → 위팔뼈 아래쪽 미끄러짐 → 불완전탈구.

🦴 2. 봉우리빗장관절 탈구 (Acromioclavicular)

원인: 어깨 위쪽면으로 떨어지는 외상. 1~6등급으로 분류 (물렁조직 손상 정도 + 빗장뼈 먼쪽 끝 자세 기반)

등급손상치료
1급봉우리빗장인대 부분 손상비수술: 팔걸이, 통증 가라앉을 때까지 수일간
2급봉우리빗장인대 완전파열 + 부리빗장인대 부분 손상
3급두 인대 완전파열, 빗장뼈 봉우리에서 완전 분리
4~6급빗장뼈 골막, 어깨세모근·등세모근 파열, 전위외과적: 인대재건술·정복

임상증상: 손상부위·팔 올릴 때 통증. 3~6급: 육안으로 보임.
물리치료: 통증 없는 범위 내 접시위팔관절 운동 + 어깨뼈 들임·내밈 + 회전운동

🦴 3. 복장빗장관절 탈구 (Sternoclavicular)

앞쪽 탈구: 어깨 앞면에 직접타격 → 복장빗장인대·갈비빗장인대 파열.
뒤쪽 탈구: 중증 외상성 병변. 신경혈관 구조 손상·압박 가능 (더 위험).
치료: 앞뒤 모두 손이나 타월클립으로 전신마취 후 정복술.

🟢 4. 위팔자관절 탈구 (Humeroulnar)

개요: 어깨관절 다음으로 탈구 잘 일어나는 관절. 소아에서 가장 흔한 탈구 관절. 동반 골절·신경혈관 손상 흔함 → 검사 후 정복 필수.

분류: 노뼈·자뼈 뒤쪽·뒤가쪽 전위가 80~90%. 나머지: 노뼈나 자뼈 단독, 앞쪽·안가쪽 = 극히 드뭄.

합병증: 위팔동맥·노동맥·자동맥 + 정중신경·노신경·자신경 손상 정밀 검사 필요.

중재: 먼쪽당김술 → 뒤침+뒤침 → 팔꿈치 가볍게 굽힘 정복 → 100~110° 굽힘 부목 고정.
물리치료: 급성기 이후 조기 능동관절운동. 지속 고정 시 근위축·굽힘구축 위험.

🟢 5. 노뼈머리 불완전탈구 (보모 팔꿈치)

보모 팔꿈치(nursemaid elbow / pulled elbow): 5세 이하 소아에서 가장 빈도 높은 손상. 머리띠인대(annular ligament) 일부 찢어져 노뼈머리 불완전탈구. 노뼈머리와 위팔뼈 작은머리 사이에 끼어서 통증.

원인: 아래팔 엎침 + 팔꿈치 편 상태에서 갑자기 잡아당김 (옷 입힐 때, 계단 오를 때 등).

임상증상: 팔꿈치 약간 굽힘 + 아래팔 엎침 위치에서 팔 사용 안 하려 함.

정복: 지렛대로 엄지를 팔 오금 속에 위치시키고, 아래팔을 뒤침하고 굽힘시킴.

🟠 6. 반달뼈 탈구

원인: 손목관절 과다펌 + 노쪽편위 (엄지두덩 방향으로 짚고 넘어짐) → 손배뼈 골절 + 손배반달인대 파열 → 반달뼈 탈구.

임상증상: 종창·통증 심함. 앞쪽 탈구 시 손목굴에서 촉지 가능, 굽힘근 압박 → 손가락 굽힘 모양. 정중신경·자신경·동맥·힘줄 손상 동반 가능.

합병증: 손배반달뼈 분리, 반달세모뼈 분리, 손배뼈·반달뼈 무혈성 괴사, 손목굴증후군.

중재: 손가락트랩(finger trap) → 손배뼈 앞쪽으로·반달뼈 뒤쪽으로 밀어 정복 → 석고고정 약 12주. 재발 빈도↑ → K-강선으로 경피성 고정 후 석고고정 권장. 불안정이나 정복 불가·신경손상 시 수술.

🔴 7. 엉덩관절 뒤쪽 탈구

개요: 엉덩관절은 가장 안정된 관절 (깊고 견고한 ball & socket). 단순 탈구보다 골절 탈구가 더 흔함. 엉덩관절치환술 후 가장 흔하게 발생. 외상탈구는 매우 드뭄.

원인: 자동차 사고 계기판 손상(dashboard injury)이 가장 많음. 굽힘+모음 상태에서 넓적다리뼈 후방 밀림.

합병증내용
무혈성 괴사8시간 이내 정복 → 10~15% 발생 / 8시간 이후 → 40% 이상. 빠른 정복 필수!
궁둥신경 마비약 10% 발생. 일부 영구 발처짐 보행(foot drop gait)
기타재탈구, 혈액공급 부족, 골절

정복 방법:

  • 엘리스방법 (누워서): 바로 누운 자세, 검사자는 굽힌 무릎에서 견고한 먼쪽당김으로 절구 속에 넣음. 가벼운 회전 도움.
  • 스팀슨방법 (엎드려서): 엎드린 자세 → 손상다리 끝에 걸고 체중으로 당김 → 종아리에 아래 방향 힘 + 발목 가벼운 돌림.

물리치료: 정복 후 2~4주부터 목발보행·운동치료. 뒤쪽 탈구: 굽힘근·모음근 구축 → 폄근·벌림근 강화 스트레칭.

🟣 8. 무릎관절 탈구

원인: 자동차 대쉬보드 충격, 운동 손상. 오금동맥 손상 가능 → 동맥 손상 주의 (2차 위험).

분류: 앞쪽 탈구 (정강뼈가 넓적다리뼈 앞) vs 뒤쪽 탈구. 혈관 손상: 앞쪽↑, 종아리신경: 뒤가쪽 탈구↑.

임상증상: 자연 정복 시 변형 없을 수 있음 (관절주머니 큰 찢김 → 체액 물렁조직으로 빠져나감).

중재: 즉각적 정복. 탈구된 뼈의 돌출 압박 or 수동당김. 구획증후군 동반 가능 → 다리 신경혈관 주의 관찰.

🟣 9. 무릎뼈 불완전탈구 및 탈구

개요: 무릎뼈는 용기사이고랑 속 적절한 위치 → 동적·정적 안정구조에 의존. 안쪽넓은근의 관여가 핵심.

원인: 보통 무릎을 부딪히거나 정강뼈 가쪽 뼈에 의해 발생. 청소년·청년기 여자. Q각 클수록 탈구↑. 구축 쉽게 일어남.

요인 유형내용
내인성 요인안쪽넓은근 약화, 가쪽넓은근 비대, 타이트한 엉덩정강띠·가쪽무릎지지띠 → 무릎뼈에 대한 '활시위' 힘
외인성 요인Q각 증가, 엉덩관절 가쪽돌림근·벌림근 약화, 발의 과도한 엎침, 넓적다리뼈 안쪽돌림

임상증상: 무릎 앞쪽 동통, 계단 오를 때↑, 휘청거림, 무릎뼈 안쪽면 압통·마찰음, 안쪽넓은근 위축, 가쪽 움직임↑.

검사: 불안검사(apprehension test) — 무릎뼈 가쪽 전위 + 무릎 굽힘 → 20~40° 굽힘에서 통증.

⭐ 임상 포인트 — 무릎뼈 가쪽 이동경로 요인

안쪽넓은근 약화 + Q각 증가 + 가쪽 구조물 타이트 + 발 엎침 → 복합적으로 무릎뼈 가쪽 탈구 유발. 치료 핵심: 안쪽넓은근(VMO) 강화

🌸 10. 측두하악관절 기능장애 (TMD/TMJD)

개요: TMD는 증상복합체질환(symptom complex disease). 만성 안면통증 4대 질환 중 하나. 성인 20~30%, 20~40세 호발, 여성>남성.

원인: 교통사고 안면외상, 치열 부조화, 나쁜 습관(한쪽만 씹기·이갈이·수면자세), 류마티스관절염, 심리사회적 요인(불안·우울·스트레스).

검사방법정상
아래턱 운동범위검사입 최대 벌림 시 위·아래 치아 간격 측정4~5cm 이상
TMJ 촉진검사손가락을 TMJ에 놓고 아래턱 머리 움직임 확인양측 동시에 앞가쪽으로 이동
치아 정렬검사입 다물고 위·아래 이 중앙 확인, 겹치는 부위 간격 측정-
씹기근육 근력검사꽉 무는 동작·씹는 동작에 저항-

중재:

  • 물리적 인자치료: TENS·초음파·SSP·냉온열치료 → 순환증진·근경련 완화
  • 정형도수치료: 관절가동술 Maitland Ⅰ,Ⅱ,Ⅲ등급 (너무 강한 힘 → 재손상 위험)
  • 머리엉치치료: 양쪽 엄지손가락을 머리엉치에 놓고 나머지 손가락은 머리뼈 가쪽면에 놓고 부드럽게 눌러줌
  • 예방·관리: 식습관·생활습관 교정, 수면·스트레스 관리, 이완운동
⭐ 핵심 수치
  • 정상 입벌림: 4~5cm
  • 호발 연령: 20~40세
  • 성인 유병률: 20~30%
  • 관절가동술 등급: Ⅰ,Ⅱ,Ⅲ (강한 힘 금지)
