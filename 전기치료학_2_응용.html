💉 의용평류 & 이온도입법

이온도입법: 직류전류를 이용, 같은 극성 도자를 선택해 반발력으로 이온을 피부에 강제 주입.
침투 깊이: 1~3cm / 농도: 1~2% / 강도: 5mA 이하 (보통 0.5mA)
약품효과
초산(아세트산, +)칼슘 침착 용해, 근경축 완화
요오드칼륨(KI, -)섬유 용해, 방부, 진통
구리(Cu+)진균감염, 국소 살균, 항진균
리도카인/프로카인(+)통증억제, 국소마취
마그네슘(Mg²⁺)경련감소, 근육이완
히스타민(-)혈관확장, 충혈, 말초혈관질환
아연(Zn+)상처치료, 염증완화, 무통성 궤양
⚡ 임상 포인트
전기화학적 화상: 양극에서 잘 발생 (산성 반응)
활성전극 : 비활성전극 크기 = 1 : 2 (비활성이 2배)
높은 강도보다 낮은 강도가 더 효과적
금기: 개방된 상처, 감염, 뼈돌출부, 감각상실부위

📡 저주파 치료 & 전극배치

저주파: 1,000Hz 이하, 역학적 효과(신경근 전기자극 → 수축 유발)
자극의 3대 요소: 자극강도 / 자극시간 / 전류강도의 변화율
전극배치특징
단극식활성전극(작음, 음극, motor point) + 분산전극(큰, 양극) → 단일근 자극
양극식크기 같은 두 전극, 근육 기시·정지에 부착, 2개 이상 큰 근육 자극
상향배치말초→중추 (음극을 몸쪽에)
하향배치중추→말초 (양극을 몸쪽에)
자극 대상특징
신경지배근시치 1ms 이내 / 맥동빈도 10~35pps → 강축 / 극성공식: CCC>ACC>AOC>COC
탈신경근시치 10ms / 통전시간 긴 전기 사용 / 극성공식: CCC<ACC<AOC<COC (반대)
단속시간비(on-off ratio): 통전시간:비통전시간
예) 10초 통전 / 20초 비통전 = 1:2
순환주기 = 맥동렬 기간 × 100 / 총 주기 시간
불편감 최소화: 4KHz - 20% / 최대 토크 발생: 1KHz - 10%
⚡ 임상 포인트
SSP(은침전극): 쌍극성 대칭파, 저빈도, 경혈·발통점 자극. 어린이 치료 적합, 권태감 적음
탈신경근 전기자극: 위축예방, 근재교육, 유착방지 목적

⚡ TENS (경피신경전기자극)

TENS: 말초감각신경(표층감각신경) 자극 → 통증 치료
파형: 이상성 비대칭 맥동전류
전극: 탄소실리콘전극(가장 많이 사용), 카라야 전극(수분 보존)
종류이론특징
고빈도-고강도겉질억제설빠른 진통, 강한 전기감, 근수축, 양극전도차단
고빈도-저강도관문조절설빠른 진통, 낮은 전기감, 습성통증 조절, 지속력 짧음
저빈도-고강도엔돌핀설느린 진통, 강한 전기감, 만성통증 조절, 지속력 길다
배치법적용
특수점 배치법경혈점, 운동점, 발통점
통증부위 배치법통증 부위 직접 적용
척수구획 배치법척수 수준에 전극 배치
이중삼각 배치법몸·허리 통증 (양극 중앙, 음극 4개 배치)
⚡ 임상 포인트
이중삼각배치 적응증: 근골격계 통증, 초기월경통, 난치성 협심증, 수술 후 통증
금기: 심박조정기, 목동맥팽대 위, 임산부
관문조절설: Aβ 자극 → SG 활성화 → Aδ·C 섬유 통로 차단

🔵 HVGS · MENS · FES

HVGS (고전압맥동전류): 이중점점파(짝점점파)
전압 100~500V, 전류 최대 300~400mA, 통전시간 50~100μs (아주 짧음)
맥동기간:맥동간 간격 = 1:99 / 변성근 자극 불가
MENS (미세전류): 1mA 이하의 아주 작은 μA 전류 → 상처 치유
세포 수준 에너지 자극 / 파형: 단상 직사각형파, 이상대칭 직사각형파
효과: ATP 생성↑, 콜라겐 합성↑, 뼈 형성 촉진, 세균 성장 억제
기기파형특성적응증
HVGS짝점점파, 높은전압, 짧은통전통증감소, 말초순환↑, 궤양치유, 부종감소
MENSμA 전류, 손상전류 회복상처치유, 통증감소
FES저주파, 신경지배근 자극기립·보행·일상생활 동작 (UMN 마비환자)
⚡ 임상 포인트
HVGS 파형을 짝으로 만드는 이유: 맥동 기간을 늘리기 위해
FES 적응증: 무용성 위축, 관절가동범위↑, 경련성 관리, 보행훈련, 어깨관절 아탈구, 일차성 척추옆굽음증

🌀 중주파 / 간섭파(ICT) / 러시안전류

ICT (간섭파치료): 2개의 서로 다른 중주파 교차 → 간섭으로 새로운 저주파 발생
· 4,000Hz × 4,100Hz 교차 → 100Hz 저주파 발생
· 중주파가 저주파보다 피부저항 적게 받아 심부자극 가능
· 변조심도: 두 파의 간섭이나 보강의 크기(백분율)
전극 배치특징
2극 배치마주보는 선상에서 100% 간섭
4극 배치전기력선 중심에서 100% 간섭, 교차점 45도 방향에서 100% 간섭효과
벡터스케닝꽃잎처럼 중심점 회전 → 넓은 부위 자극 (치료부위 불명확 시)
6극 배치조직을 입체적으로 자극
현상설명
길데마이스터 효과맥동이 모여 가중 → 신경흥분, 탈분극 (상대불응기에서도 가능)
웨덴스키 억제1,000MHz 이상 계속 자극 → 재분극 기회 없어 흥분성 완전 상실
보강간섭골과 골, 마루와 마루의 중첩
상쇄간섭골과 마루의 중첩 → 합성 변위 약해짐
러시안 전류: 교류전류 돌발시킨 시간 변조 교류
최대 수의적 수축과 같은 정도의 근수축 가능 → 운동선수 근재교육·근력 증강에 주로 사용
주파수생리학적 효과
100Hz 고정파미세진동, 혈관확장, 진통, 교감신경억제
1~10Hz 고정파운동신경자극, 근수축 유발(정상근)
1~100Hz 리듬파미세진동, 조직 활성화↑, 부종 경감
90~100Hz 리듬파혈관확장, 진통
⚡ 임상 포인트
웨덴스키 억제 해결법: 주파수↓ / 리듬감 부여 / 진폭변조 주파수 사용
역동전류(베르나르전류): 교류 정형파를 정류한 정형 직류. 완파장·단파장·짧은교대파·긴교대파 4종류

🔥 고주파 치료 (단파·극초단파)

고주파 특징: 100,000Hz 이상 → 이온 운동 거의 없음 → 열효과만 존재
⚠️ 고주파는 주파수 낮을수록 심부 침투 (저·중주파는 반대: 높을수록 깊이 침투)
열 발생 순서특성
유전율 높을수록 열↑혈액 > 근육 > 뇌 > 지방 > 피부 > 뼈
비저항 높을수록 열↑뼈속질 > 지방 > 피부 > 간 > 근육 > 혈액
구분단파치료극초단파치료
주파수27.12MHz2,450MHz / 915MHz / 433.9MHz
가열심부가열표면가열
치료부위양쪽 치료 가능한쪽만
범위넓은 부위좁은 부위
단파 전극사이거리: 2~4cm 유지 / 가까우면 표층열, 멀면 심부열 집중
단파 전극배치법: 대면법 / 공명법 / 교차법 / 병렬법 / 콘덴서패드 코일법
극초단파 선택적 가열: 물(고함수조직) 많이 함유할수록 가열 잘됨
극초단파 도자 종류
· A·B형: 1극 도너츠형 빔 50-100-50 → 볼록하고 불규칙한 돌출부
· C·D형: 2극 타원형 빔 100-75-50 → 오목하고 좁은 평평한 곳
· E형: 2극 타원형 → 오목하고 넓은 평평한 곳
· 반터널형: 신체 양쪽 동시 가열
⚡ 임상 포인트
고주파 공통 금기: 악성종양, 허혈부위, 금속삽입부, 심박조정기, 결핵
극초단파 추가 금기: 남성생식기, 비만, 성장기 뼈·관절
맥동단파투열: 비열적 효과 목적 / 적응증: 염증·타박상·혈종·윤활주머니염·정맥굴염

🌊 초음파 치료

초음파: 음파 진동 (금속 적용 가능)
주파수: 0.5~5MHz / 1MHz·3MHz 많이 사용
발생 원리: 역압전효과 / 전달에 매질 필요
특성내용
침투 깊이주파수 낮을수록 깊이 침투 (1MHz가 3MHz보다 더 깊이)
흡수율 순서뼈>연골>힘줄>피부>근육>신경>지방>혈액
반사율도자-공기 99% / 뼈-골막 70% 반사
강도0.25~3W/cm² 주로 사용
적용 시간연속 3~10분 / 펄스 5~15분
적용 방법: 50% 겹치기, 2.5cm/s, 90도 유지, 젤 충분히 사용
고정치료법: 1W/cm², Trigger point / 직각 고정, 국소 0.8W/cm²
수중치료법: 23~34℃ 증류수, 1~2cm 떨어트려 고정 적용
음파영동 vs 이온도입
· 약물 극성 불필요 / 분자량 크면 어려움
· 이온도입보다 더 깊이 침투
⚡ 임상 포인트
공동현상: 기계적 효과로 조직의 액체 안 기포 발생 → 조직파괴
도자를 이동시켜야 하는 이유: 도자 가까이 열 집중 → 화상 예방
금기: 뇌·척수, 눈, 생식기, 임산부, 악성종양, 방사선 치료부위, 노출된 신경조직
