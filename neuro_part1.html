🧠 1. 정신상태 (Mental Status)
1-1. 의식수준 (Consciousness)

이완 상태 → 자극 반응성을 기준으로 5단계 분류 (ADSSC 암기)

단계핵심 특징
명료 Alert정상 의식, 자발적 움직임, 시간·장소·사람 인식
기면 Drowsy수면경향, 큰 말소리 반응 가능, 비교적 정확한 대답
혼미 Stupor강자극 시 약간 반응, 이해 어려움, 반복강자극에 일시흥분
반혼수 Semicoma자발근육운동 거의 없음, 꼬집으면 피하려는 반응
혼수 Coma강자극도 무반응, 눈 감겨있음
주의력 정상반응: 앞으로 7개 숫자, 뒤로 5개 숫자
지남력 손상 순서: 시간 → 장소 → 사람 (회복은 역순)

1-3. 인식 / 행동 (Gnosia / Praxis)

인식불능증 (Agnosia) — 감각 정상인데 인식 못함

  • 시각인식불능증: 5가지 물체 중 4개 명명 못함
  • 색체인식불능증: 물체 색 인식 불가
  • 입체인식불능증: 촉각으로 물체 인식 불가

행위상실증 (Apraxia) — 마비 없는데 과제 수행 불가

  • 운동: 연속동작 어려움
  • 관념: 실제 사물 사용 불가
  • 관념운동: 구두명령·모방 불가, 실제 사물 사용 가능 ★
관념운동행위상실증 예시: 칫솔로 양치하라 명령 → 못함 / 몸단장 중 칫솔 들면 → 양치질 함

🏥 6. ADL 평가도구
수정된 바델지수 (MBI)
  • 만성질환자 ADL 자립도 평가 / 총 100점, 10항목, 5단계 척도
  • 항목: 개인위생·목욕·식사·화장실·계단·옷입기·대변·소변·이동·보행(휠체어)
완전의존 0~24 심한의존 25~49 중등도 50~74 경도 75~90 최소의존 91~99
FIM (기능적 독립측정)
  • MBI + 의사소통·사회적 인지 추가
  • 신체 4영역 + 인지 2영역, 18세부항목 / 18~126점, 7점 척도
  • 재활치료 후 예후 추정에 효과적
Katz ADL Index
  • 6항목(씻기·옷·화장실·이동·배변·식사) / 6~18점, 3점 척도
  • A(완전독립)~G(완전의존) 누진점수
  • A가 먼저 독립: 식사→배설→이동→화장실→옷→목욕 순
PULSES 프로파일
  • 신체기능 + 지지요소(정신·가족·사회·재정) 평가
  • P·U·L·S·E·S (6항목) / 6~24점, 4점 척도 / 12점↑ 심한장애
SCIM (척수독립척도)
  • 척수손상 전용 / 자조관리(20)+호흡&조임근(40)+이동(40) = 100점
⚖️ 7. 균형 평가도구
BBS (버그균형척도)
  • 정적+동적 균형 / 원래 낙상위험 측정 목적
  • 14항목, 5점 척도 / 총 56점
  • 41~56=낙상위험낮음 / 21~40=중간 / 0~20=높음
FRT (기능적 뻗기검사)
  • 노인 낙상·균형 평가 / 지지면 고정 → 팔 최대 뻗기 거리
  • ≤15~17cm: 균형 제한 / ≥25cm: 균형 좋음
TUGT (일어서서 걷기검사)
  • 앉기→3m→돌기→다시 앉기 시간 / 비마비측 축으로 회전
  • 10초↓=정상 / 11~20초=허약노인 / 14초↑=낙상위험 / 20초↑=이동제한 / 30초↑=의존ADL
POMA (Tinetti)
  • 균형(9항목·16점) + 보행(7항목·12점) = 총 28점
  • 25~28=낮음 / 19~24=중간 / 19미만=높음
🚶 8. 보행 평가도구
10MWT
  • 총 14m 중 가속(2m)+감속(2m) 제외, 중간 10m 시간 측정
  • <0.4=실내 / 0.4~0.8=실외제한 / >0.8=실외가능
6MWT
  • 평평한 면 6분간 최대 보행거리 / 심폐기능 객관적 평가
  • 최대하 운동검사 (환자 능력에 맞춤)
FAC (기능적 보행지수)
단계내용
0보행 불가 / 2명 이상 도움
11인 지속적 지지
21인 간헐적 도움
3신체접촉 없이 지시·관찰
4평지 독립, 계단·경사로 도움
